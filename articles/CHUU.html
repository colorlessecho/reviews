讀書劄記 @ Colorless echo -the days never back-

努力的話，一小時內可以解決的遊戲。不過，我還是花上了三四小時，畢竟要完全熟悉一個 game 的模式、搞清楚是否有其他おまけ或暗門啥的，多少得花些時間。
OP 還搞得有模有樣的，沒想到竟是影片。沒差，反正玩的時候和那無關。基本上這 game 並沒啥分歧，並且完全沒有累贅選項，（大進步！…不過出現在這樣的 game 中實在不怎麼感心(笑)）所以不需要攻略。照 Option 項目中每個分支走一走 CG 就全出來了。就算懶得這麼做，最後幾十個位元全填 FF 就可輕而易舉看到所有事件及 CG。
系統太過緩慢，skip 有點累；雖然明明就沒啥劇情。鍵盤不支援，更令人不滿的是 Ctrl 鍵 skip 及 review 都沒有。大致說來，使用中令我想到 ExcelSaga 中的美女劇場——簡直一個模子吧，一樣爆笑，尤其是最後那個音效。那個 St.啥的博士算是啥東西啊，真是無所不在，連別人家中都可輕易闖入。和渡「邊新」一有得拚。比較有趣的還是美女劇場，但使用中就有那種感覺。
不，雖然有許多爆笑性的色色場面，這絕不是 R指定（笑）。
