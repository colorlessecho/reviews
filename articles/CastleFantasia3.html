キャッスルファンタジア(Castle Fantasia)～エレンシア戰記～
花了不少時間把有的沒的（包括好感度）改出來了。遊戲除了進行中與攻略的七個女孩交談外，結局全在 MEMORIAL 中：每個女孩兩段，各在最後戰前及最末尾。都沒泡到時戰前秀一段告訴你大家各分東西的訊息，最末尾就省略。知道這點後，雖然能力都已加到最強，我還是放棄了，直接看 MEMORIAL 比較快。反正在跑過一次後就只剩下 CG 啥的收集，既然已經改出來，那就甭白費工夫了。
劇本寫得還不錯（搞笑段落夠多），不過整個回頭看一次，卻覺得相當普通——世間劇本，八點檔不都是類似的東西嗎？倒不如說，大部分的美少女ゲーム因為題材配合不好，許多劇本作家又限於當下大環境而沒能有創新，因此表現反而不及這樣的東西。話說如此，這對不時常玩 RPG 的我來說，還是個頗「好玩」的 game，不然我不太可能還有興趣連兩天凌晨三四點才睡，並好好跑過一次。當然，想要把有的沒的改出來之企圖心恐怕也是原因之一，為了改圖，事實上為了這一次重複跑過（不是攻略新女主角！）不下三四次。劇中還有不少長篇大論，當作沒看到就算了。那國家民族啥的，連反方的意見都聽到要爛了，亦即那只是泛論，了無新意。
畫自然不用說，山本阿姨社長不是當假的。那隻雀…在山本許多遊戲（包括 MAW）中似乎都會出現可愛鳥綱腳色，是他的癖好嗎？音樂雖然非我所喜歡的風格（連配樂都像八點檔在用的-_-），但聽起來不錯，整體應可算是不負我期望了。系統大致還好，沒有 review 是比較可惜的，另外用全螢幕時偶爾切換回 windows 再切回去 process 會掛掉消失不見（到 v1.02都一樣）。因為沒各女主角的結局 save，所以就不擺 save data 了。（雖然是無敵 save…）戰鬥時，假如目的是打倒某位將領，請直接派テルル功過去即可，省時省力省大腦（笑）。更好的方法是見識過幾場後就把本 game 當作 ADV 玩（爆）。方法：Save99.dat,290h=1，如此即可選擇 skip 戰鬥。其實，Save99.dat 全改成00 00 00 01，將會有您夢寐以求的效果，更加省腦力。
忘了說，某人一直令我回想起戀ごころ，那絕對不是錯覺。
