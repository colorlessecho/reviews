變更地區及語言成日文 (日本)
您可以試試：
  • 請打開控制臺「地區及語言」，將系統預設的顯示格式改成日文。
  • 自原先的安裝資料夾 copy Save 目錄至安裝目的地。

Trial and error 經過

雖然聽說かぐや已經失去昔日的光彩（いや、這不是重點。），但還是興味本位で安裝了姉母娘でイクッ！。執行後卻發現，標題變成亂碼也就算了，要開始本文時就 crash 了。因為 system locale 一直都設定成日文，加上本遊戲特別強力放送 DirectX 的訊息，因此估計大概是 DirectX 語系設置問題。打開 dxdiag.exe 一看，System Information 中果然顯示使用的 Language 是 "Chinese (Traditional) (Regional Setting: Chinese (Traditional))"。這種挑執行環境的遊戲也不是第一次了，之前 minori 系的，如 ef 也頗為狠毒。但改成日文顯示介面、持續改變環境重裝了幾次 DirectX 11 依舊無效。查了一下，希望 DirectX.cpl 能改變語系，都裝上了 Microsoft DirectX SDK (June 2010) 才發現還是沒法解決。再次打開控制臺「地區及語言」，想說「難不成還要重灌系統？」那就太大費周章了，決定死馬當活馬醫，一個個將之改成日本設定試試。才測了「格式」就發現 OK 了。之後發現此設定並非即時性的獲取，因此若有需要，開始執行後便可改回來。另外，遊戲程式似乎會將首次執行時的部分環境紀錄於 save 檔中。為了正確顯示，請在改變環境後，移除原先的 Save 資料夾，避免像是主人公的名字變成亂碼的問題出現。

原理

DirectX 的 Language，包括對非 Unicode 編碼的解讀，採用系統預設的顯示格式。雖然我以為這是有問題的設計理念。照理來說，整個系統的介面應該都採用「顯示語言」的設定才是。但除了 DirectX，似乎也有其他像 Google Chrome 採用了這樣的設計。或許是因為「顯示語言」這東西僅在英文版的 Windows 才看得到，因此不得不作些變通？
無論如何，アトリエかぐや的遊戲系統，顯然連在以「API 描画」起動時，對於文字的讀取還是透過 DirectX 獲得；這造成即使設定了 system locale，也不會按照這設定取得文字編碼。為了解決問題，只好改掉顯示格式之設定。

姉母娘でイクッ！

女たらし vs. 上手い男の決戦？簡單的說，就是普通寝取られもの的顛倒吧。這次不但站在寝取る的一方，且越看到後面，越有之前那類作品中第三者（男）角度的感覺，使出了許多只在遊戲中才看的到的陳腐題材。不過 H scene 頗濃い。使える。

故事本身長度中等偏短，對現在的我來說已經足夠了。ライトノベル作家の主人公久高有史喜愛的親姊姊葛城依理子，高中時就與老師葛城光男奉子成婚了。當時給了主角很大的打擊，但現在卻因為丈夫風流成性，這次又上了位女大生桐乃木奏，而一氣之下帶著女兒葛城碧璃離家出走。因為不好意思找其他人，就輾轉到了主角家。
依理子個性樂觀不拘小節，碧璃則因為父親的不規矩，因此對男性很是不信任。話雖如此，本身卻對性相相當有興趣，積極進攻男友。光男一副女たらし樣；雖然對腳踏兩條船的狀況感到棘手，在奏面前卻又志得意滿。這邊的設定基本上與官網公布的相同，但要說其合理性的話就沒那麼真實了；在故事進行中，我一再折服於這兩母女的性感。まあ、不是這樣的性格，主角怎有機會趁虛而入呢？
主角先是藉故滿足姊姊的性慾，之後更計畫要直接寝取る。在不同路線中，可以選擇是要姊姊、姪女，或是乾脆連奏都一起搶過來。最後總有一幕讓光男後悔莫及的 live 春宮秀，然後這使い捨てキャラ的下場就沒人管了；此番處理正是使我不得不想起之前眾多寝取られもの之處。

主角的武器除了イケチン+テク男之外，還很會稱讚女性。女人天性就喜歡這種褒法，因此很快就落入主角下懷。本來認為主角這種打一開始就沒安好心的想法實在不好，但到了其與姊夫意外碰面那幕，狀況又不同了。依作者言語間透露出來的訊息，這傢伙也不是啥好貨；特別選了些表現不知節制的話語，顯現這花心大少…花心大叔並沒真正的覺悟。
不說姊姊的是近親相姦，連姪女碧璃的路線都有些牽強。碧璃男友本身就夠怪了：這青春男孩被所愛的女友襲來，竟還露出厭惡的表情；此般男孩似乎不太現實啊。而主角雖然一開始還算頗紳士，倒是碧璃竟然還拗不過自己的性慾，主動要求！只能說此位ツンデレ小姐實在是過於悶騷了。無論如何，我還是不認為主角只因為男朋友過於閉思，就可以上了碧璃。這樣的做法，在現實中通常只會造成其不幸罷了。另外，初体験一般不會先後再前吧，這樣太不衛生了？
在到中段時我都還用得頗愉快。不過對那幾幕春宮秀則感到未免過於做作。依理子的好歹還有個理由，雖然我覺得或許可以不用這麼難堪；但另兩場不但沒啥必要，還幾乎只是重現一般的寝取られ濡れ場啊。
至於最後的結局，怎麼想都不是永久的。連長期也說不上。因此，簡單一句話就是玩家爽就好了。別想太多吧。
本作 CG 還頗華麗細緻，相較於一般遊戲，有點 Tony 的質感。在細節上不能說完美，舉例來說我還是比較偏好山本大媽…失禮，應該說山本妹妹的輪廓畫風；但能做到本作這程度確實也不錯了。 BGM 普通，声優表現還不錯。系統大概就前面提到的顯示格式問題，其他還好。 整體說來，不說劇本，光是當作抜きゲー來說，還頗實用的。
