讀書劄記 @ Colorless echo -the days never back-

(1998)三月十八日，架構大致完成。幾天來一直在電腦教室⑵奮鬥著，連報告都沒心情作。並不完全是瘋狂投入之故，自己的 homepage 能上 WWW 總是令人些許興奮的。
3.19 以無意義而可笑的舉動詢問他人，好的網頁應具有如何的條件。No one has ideas.最後，答案自然是幾天來預設的，加強內容吧。
3.20 將 homepage 大致做了個收尾的工作。由於 What I see isn't what I get, 還得為每一種瀏覽器的各種版本量身定做；幸好只有 Netscape 及 Explore 兩種。幾天來在電腦教室⑵的努力算是有了些成果。剩下的，就是 addition 的部分了。
3.22 十幾天來無所事事的生活大致結束了。未來希望會有好段時間不會做改變，回復“平靜”的生活。這段時間內，這個站會像死了般。本來就是個沒多少人來的地方，只是回歸其本質罷了。一如頁面上所言，尚未完成；不過，已經好幾天沒讀書，覺得沒什麼閒工夫可以再做下去了。反正一個 homepage 未完成也不是什麼大不了的事，那些標榜“持續更新中”的，不甚至是永遠未完成嗎？
Homepage 這種東西的用語原該是愉快的，大概是我對公開自己的任何想法都不能處理得很好，之後就請各位多擔待了。每天上來做報告也不是什麼體面的事。
3.26 或許是最近太閒了，沒事還是往電腦教室⑵跑。或者是一心想著許多夢想的大志，將 CGI, JAVA 等學會後用在自己的 page 上；結果沒半樣做到，心覺不甘吧，起碼加點 JAVA Applets 也好。就這樣，page 上就多了幾個 applet。
3.28 Page 中的物件持續增加中，感到最近這一頁的完全開啟越來越慢了……
3.30 最近又遊走於數目眾多的 homepage 中，上次是半年前的事了。拿著一本 1996 出的書，十個有兩三個找不到，其他則多數改了位置。（電腦會自動連去，或是一個請我好心告訴 the source，以更改至最新位置的告示。）尤其是 .edu 的，幾乎沒一個還在；想必他們在稍有成就時就出校門了。INTERNET 上資料的更新是瞬時的。
4.1 這裡的人特別記得這個日子，好像一年中只有這天能作弄人，是不可失的機會。不知道幾年沒憶起過所謂“愚人節”了。要不是有人提醒，就會像鄉愁一樣，是個忘不掉、平常不會想到、每當縈繞腦海卻又感覺特別深的名詞……
4.2 發現了個用了“DHTML”的站 小豬的窩，正研究中。（注：已死，不過 Internet Archive 還有。）
4.3 七八天的春假，網頁七八天的停擺，一禮拜小說及卡通、電視的進修……
4.12 之前問過一些同學，多數對這個沒甚麼用的站其特效還覺可以。雖然是個證明“我存在”的小書籤，這也許是最大的恭維了；但若是有野心些，那被如何的讚賞也不算甚麼。畢竟，這學校的學生不以此為業。唯因沒多大才幹，弄出個主題有些困難。加了點 link，製作首頁的心得；最後可能像大雜燴一般。
注意觀察會發現這個站至今用了非常多的 Buttons，卻沒有半張 Image Map；其實是因為懶得去畫那張圖。自己去設計得花一下午，況且這個站一直未完成，說不定哪天還要新增甚麼項目。
4.17 現在的工作是將以前的 collections 轉成清楚地標示，有時整天泡在各網頁間還真累人。
4.19 期中考了，突然對作 homepage 這種無聊的事感到毫無意義……
4.20 將 object window 中設定 screen 的功能加入主頁。
5.1 每每檢視網站時聽到 wav 或是 au 的聲音和正在聽的 mp3 混在一起就很痛苦。想到別人在逛這個站時若也遇到這種情況……也許是不加音樂的另一個好理由。
5.3 又是完全不值票價的一日，突然發覺在 WWW 上，有什麼好說的嗎？Homepage 這種東西，其價值越來越令我懷疑了……
5.9 在 yes.net.tw 找到了 layer 的用法，昨、今一整日都在改 work 的原始碼。
5.20 好像好幾天沒登錄這一頁了。將近十天來一直在研究用 C 寫 CGI 的方法。不多述了，要是某些真正的初學者有不能突破的瓶頸再來找我吧。總覺得最近被作業、考試及報告等壓得喘不過氣來，更糟的是這並非我的興趣……
5.24 經過了兩三禮拜不眠不休的奮鬥，今天終於把 C 做的留言版送上 page。有任何意見請記得告訴我，沒批評就沒改進。不過，這留言版尚不符合我的期望，只是一個濫竽充數的作品罷了，很多想要的功能都尚未加入。還是一句話，請多多包涵了。
6.12 對此島是否有募兵的必要，我仍存有疑點。しかし、無駄だ。太多的事已失去將之改變的意志，just let it be，想執著什麼嗎。
期末考前的這個時候才想起要作個記錄，我時常做這種事。因為將來臨的壓迫，心突然便澄澈了。當然不一定是真的，這很難解釋；也許我的心一直都是這個樣子。
9.14 沈寂了上百天，網路世界又有相當大的變動。電腦教室的軟體似乎做了翻整修，可能連工作站的都有變更；上學期作的程式竟不能編譯，看來又得多加努力了。最傷腦筋的是一堆 link 都改了，體會到了維護網頁的困難。
9.15 原來是換了新的系統，將執行 CGI 的權限取消了。（唉、為什麼感到自己的人生是如此多災多難呢……）一位以前的同學說的好，現在做網頁也沒什麼好講的。也許，過不久會把這個網頁關掉吧。還沒提到上成功嶺的事。一些感觸常在事過境遷後即失去了再述的心情。縱然激懷滿胸，終是敵不過涓涓光陰。不過，當時曾立志一定要寫些什麼，好歹在成堆筆記中挑幾句出來吧。「軍中生活最需要的是只會服從，不會想太多的強力機器人。」這是他們的理想，但若是以普通人類之身分去當兵，最佳的適應法是“有點小聰明，對人生、哲學等完全沒興趣思考，卻能迎逢上司的心；”雖然還不嚴謹，但我想成能力作為英雄的小兵，能有這樣的生活最能適應了。那種方程式般的生活，存在於日常的模式之下，失敗後就沒有回復機會，偏見的（軍人常僅從一面看世界）、表面化之生活，連品質的改善都需要無法想像的力量才能實行。空殼一般，相當沒效率之地（也許就因如此他們才講求效率吧）。可惜我從不是那樣合適的軍人機器。不能形容是恐懼或是絕望，這實質上完全合乎先前想像的二十五天，卻似乎像昨日天上飄過的一片雲般。
10.2 好久沒記這頁了。學校裝了 VB5，這禮拜來沒一天不泡在電腦前的。今晚學校與外界又斷線（工作站當了吧），沒人來修，（說不定重開機就可以了？這我不敢說。）正好趁這時間補回一點東西。
兩三禮拜來大致以熟悉 VB 為主，雖然有進展，不過實在不多。只是以現在的心境，要像從前致力於此也不大可能了。現在的心境，舉個例子；前不久還想對成功嶺大發言論，不到一個月後的現在已沒那毅力了。幸好老早就知道自己，已強迫的記錄於日記了。即使是日記也一樣，不督促的話，一天可能兩三行就解決了。這看起來沒什麼，但再想想，人生又過活了一日，遺留下來的竟只有一兩百字，未免太悲哀了。有日記還好，沒有的，普通的人生根本就連逝水年華都稱不上。日復一日，回想起來一點印象也沒有。我很不喜歡舉例，小學、中學怎樣等等；就這樣吧。
星期一中秋，同班同學連星期二早上一二節都調好了；即使不一定找得出補課時間。是因為教育心理學不好調，否則那天早上都不會有課。……算了，我就是過於不識時務，才會落到這田地。現在趕趕報告吧，一些自己的作業，就像寫 HTML 及 VB 程式般。
10.13 昨天聽到系上通訊錄，又想到做個 CGI 磨練磨練自己；於是借了工作站空間，將 page 搬到這了。幸好本來就是個可攜度相當高的組織。至於為什麼要這麼麻煩，其實是有內情的……
11.26 這幾天仍然是天天和 CGI 對抗著，只是為了使資料結構更簡單。
12.4 突然發現用 IE 瀏覽此頁簡直是受罪，連忙修改了一番。不過，一些特殊效果還是得靠 Netscape 才行。
12.8 又修改了一些程式。前幾天似乎有用查詢功能的，當時若有奇怪的情形，已修改完成，僅致上歉意。唉，每天的作息就在改 bug 而已……
12.9 帶了 92.2 KiB 的歌詞回宿舍，其中有二三十首要翻譯的吧……

12.30 資料越積越多，現在執行一次平均都要半秒鐘以上了……本來只想將一些自己有興趣的東西搬上來，順便練練日文（就因知道自己還不行）。結果數百首只譯了七八分之一，還有不少是自己都沒聽過的。

About 言論及爭論

今天網路情況不太好，隨時有斷線的危險。（已經斷三四次了）
	就簡單說說吧。

在這個版，原先是想營造出比較穩重的風格，
起碼別太輕浮，或是像其他某些版一般，上 BBS 就專門打發時間用的。
當然，既然 Yesme 兄已申請作板主，就看 Yesme 兄的作為囉。

首先應該談到的，應該是言論方面的問題。
我想，各人生活環境不同，有的平步青雲，也有平凡過一生的。
這不是 complain 或 fuck 版，因此希望每個參與者都能秉持較公正的態度，
即使有覺得不正確的，也不必太劇烈的反彈——委婉的提示即可；
且各人言論，其實都尚未達到能稱作“盡善盡美”的程度，
因此，希望這個版不要出現尷尬或是謾罵的場面。

About 主題

高中老師曾提過，有三種話題最好小心討論，
其中兩個是宗教及政治。（第三個是民族？性？軍事？）
其實，只要是每個參與的人都投注太多的主題，
就很容易出現混亂、爭執及各說各話；
BBS 上不能打架，所以就用罵的。
不過，在下倒希望能對這些價值、觀念等作出討論。

但是，我一直以為，世界上之所以有如此多價值觀，
還有“沒一個標準答案”等，其實是因為
在這方面只要一種思想能符合“自圓其說”及“能使眾人接受”
 （各位若有其他見解，可 add 或再提出。）
兩項，都能算是一種答案，所以才沒個標準。
在這裡，除了對一些名詞的正名外，
希望也能產生幾個這樣子的陳述，我們稱之作“真理”。

哲學書籍一向被認為深奧，其實常人也很少思索到這方面的問題。
包括政治、宗教、人存在的本身及社會現象的本質意義等，
但我覺得，有很多道理其實並不用以如此高深莫測的方式述說，
 那樣只是徒增困擾而已。
用日常言語，也可達到使人明瞭的效果，
這是我希望在這個版能看到的。

我不鼓勵大家常上，
有些事常常得事過境遷、蓋棺論定後才雨過天青，
多想想後發表的言論總是較嚴謹的。
（不過，有時還真擔心最後會不會變成只有我在 post ……）
爭論難免，希望大家多克制；
另外，可能的話其實開個匿名 post 的權限也不錯；
畢竟有些話不太好講，不過勢必會出現些濫文章，
這方面就有勞板主砍掉不當言論了。

（嗚……一小時內斷線第七次了，會被扣多少啊……）
這樣，對這個版來說，會不會負擔太重了？

電話

「請找……。」
「呃……抱歉，他不在。」「呃……他剛剛出去了。請稍等，我去叫他。」

從不知道何時起，我就重複接著不是給我的電話。
相同的語氣、相同的構句。

不，其實我知道的。一開始就是那樣子。
仔細想想，這是很公平的，就像是一樣米養百樣人，各取所需一般的自然。
只是，接久了自然多少會有點厭煩，尤其是那種明明知道這次要找的不是自己，看著那電話悠閒的鈴聲；話筒就在身旁還好，拿起來仍是一樣的語氣、一樣的構句；有時卻呆坐著等待一切恢復平靜。四聲、五聲。即使將心擺在其他地方，我總是能在心中暗數著。是本能吧，不過這種本能到底有什麼用，我也搞不清楚。
通常算到八九聲電話就斷了。這大概也是一般人能忍受的長度。偶爾也有長達二三十聲的例子；「這樣的人不外乎情人，或是家中天性純樸的母親吧。」一邊等著，我一邊想。
最長的一次，足足響了四十七聲。那天心情還好，我在大概第三十次就遲疑著是否該去接了。不過看來對方算得很準，才剛拿起話筒，「koro」就傳來那僅僅能聽到的「嘟～嘟～」不停迴繞。聲明終於放棄的象徵同時帶給了我倦怠感，我緩步爬回床上。

不知為什麼，我從未想到要去等待自己的電話。或許是因為，那根本如同在十二點就會消逝的南瓜馬車一樣的動人、卻不切實際。或者更切實的說，是走在路上被招牌砸到一般，若有還無的痛心。但我偶爾仍會接到家裡及其他不知名的人打來的電話。說「不知名的人」並非誇大其詞。
「是……學長嗎？我是學妹」陌生的音高，陌生的語調。「明天我們想借環保社的那塊紅布，聽說您是環保社的，能幫我們借借嗎？」這是誰？「啊……抱歉，我不是環保社的。」「嘿？奇怪，是學治會會長跟我說的啊。」為何打電話給我？「啊、是，我認識學治會會長……」「那你能幫我借環保社那塊紅布嗎？」到底想要什麼？「啊，那就拜託了。」尚未見過面的交談？
「啊、明天我們有集會，你要來嗎？」不是之前拒絕過的……嗎？「呃……我……」雖不擅長，並非不會；只是不太想說謊的我認真考慮著該如何回答。「明天有事……」「什麼事？」「……」糟了，不掰不行嗎？「好嘛好嘛，來嘛。明天大家都會到的喔。」我真的說過，不是贊同你們，才參加你們的啊……
不外乎這類事情，從一開始就持續到現在的；我從不能理解對方是誰，僅僅是聽著那一側的某人，喃喃唸著不知道是不是我的名字。

在經過國中、高中後，我仍舊沒有改變，除了放棄得更徹底外。嘴巴上成天掛著絕望的話，但明明沒有什麼好爭的，卻是一副叛逆的模樣，好像將打敗別人列為天職。就算如此，電話仍然是所接觸過最難纏的怪物。接著不是給自己的電話，即使是，也不能辜負他人的好意、不知不覺中，我開始排斥起電話。每響的鈴聲都有如喪鐘般，我深怕接到一封封的訃文。當然還沒有到那種程度，不過我所懷抱的恐懼，不是一般人所能想像的。幸好還沒有人問過我：你為什麼這麼討厭電話。
電話中的人們總是充滿著朝氣，不論是哭著笑著。真的到那種不能保證下一口氣的時候，大概也就不會再舉起話筒撥號了吧。而那朝氣一直威脅著我。不論是請求、邀約或是命令；我就這麼的過了數十年，還是沒能擺脫掉那難以忍受的苦痛。「迷惑」，的確是這麼回事。

電話仍舊會打來：只是，我知道終有一天會變成這樣子的。
「我要找的就是你。」
「抱歉，我已經不在這了。」

愛滋病在十年前還是不治之症。即使在現在，情況也沒好多少，只能希望拖一天算一天。得這種病嚴重之處除了知道自己無藥可醫治外，得知自己有病在身的人還會怕被感染而能躲則躲。這是較誇張的情況。但是對未知事物，尤其已經顯露出其危險性的，要人不避之唯恐不及不是容易的事。當真的有這樣子的病出現，先跑的人或許大多是醫生。
愛滋病的情況還好，若是提到黑死病，那就真的顯露出人性現實的一面了。那時的人只知道接觸得病的人，甚至同室呼吸就能得病。十四世紀的流行使歐洲的人口減少了四分之一，可知當時人心之惶惶，只要家中有人有徵候，就十萬火急先送到自己都不知道哪裡的地方了。
因為對愛滋病較瞭解，並且除了體液的交換等幾乎不至於感染，所以才能提到愛滋病患的心理治療。假如光是日常相處都得穿防菌衣，那治療起來還真是相當頭痛。現在有的療法，最著名的要算是「雞尾酒療法」了，而卻也不能完全根治，所以愛滋病患之心理治療也相當重要。除了類似得癌症般人生只剩下數年，甚至連時刻都定好（就像等死般）外，由於得此病者許多是因為性交所致，也特別令人另眼相待——即使事實上是因為其他輸血等等的緣故。
除了病因使病患可能感到羞慚外，若因此病拖累家人，這些都是容易影響病患心裡的因素。對這些已經絕望的人來說，讓他們好好地走完人生大概是我們所應盡力做到的吧。
當然，家中若有一個愛滋病患的確是不好過的。不過應先使其知道，我們（家人）並不因為他的病需時常麻煩他人而嫌惡他；也不因無處不在的偏見（若是有的話）而排斥他。若是病患的心理也調適不過來的話，那還得和顏悅色的解釋。有時這樣的病患就像任性的小孩般。總之，這是得花很多愛心的工作，甚至治療者或是相處的人都會承受不住。

2009/8/30

一九九九，最後一夜

來源文章選自張老師1999.12月號 46-49

在西元從 1999轉變到以2為首年分的這時刻、這個日子，相信對大多人來說多少都有些特別：甚至大多數人都是以狂歡慶祝來度過1轉2的那一秒。或許是如此的充滿欣喜，但是也正好視為數不少人反省的時機。人總是這樣的：面臨一個重大的日子，當自己剛好心境澄清時，就容易想到要反省自己。面對新千禧年的來到，舉世歡騰的這一刻，或是稍後慶祝活動結束後，心靜下來的人們，不知不覺就會想到：「這新時代的來到，我自己是否也有新的改變，或是新的應變方法？」「新時代來臨了，未來的人類終歸何處？我們能做到的，到底又是什麼？」「相對於過去的自己，未來我是否能有新的突破？」這些都是很正常的，因為新千禧年的來到同時也暗示了一種「新」的意味。面對不知如何應對的未來，不知所措的人們多少總會徬徨。
在這篇文章中，首先見到的是一些人慶祝的方法。「一九九九，最後一夜」，除了因為 921大地震而不能成行的玉山倒數團外，大多數的人都是和朋友、家人或是男女朋友一起；整體說來，雖然個人有各自的方法，但舉的例都是以些特別的方式，和自己最重要的人事物一同度過。雖然也有其他的慶祝方法，但在這樣的日子中，大家畢竟喜歡尋找溫情。
不過，這是三十歲左右的人作的文章。年輕人呢？對年輕人來說，2000年的來到到底代表了什麼？身在這樣一個變動的時代，其實我們都已見怪不怪了。新年假期像普通的週末，春節是短暫的年假，以及偶爾不耐煩的應酬；連平常的假日也幾乎和上班上課的日子沒啥兩樣了。我們在乎的，不再是過去那些人所珍重的；所以乍看之下一如我們什麼都不在乎一般，更進一步就是隨心所欲、只要我喜歡，有什麼不可以。教條？漸漸地，有些人正在脫離束縛中。親情？在人與人間交情淡如水，每個人都只是個獨立個體的現代，往昔我們所珍重的，而今也不再一成不變。所以，我們越來越徬徨。人總是需要依靠，但當有一天沒有東西可以抓住，此身正如井中墮下的石頭時，身為人所具有的焦躁就占據了心靈。事實上，也正因為是這般的無助。就算知道是如此，我們什麼也作不到。
當然，大部分人還不會想到這一層，只是茫茫然照著過往的方式生活下去：鞏固自己與親朋好友間的感情，託付己身於工作及友人間，無聊時讀讀小說，看看電視。想得太多還會被認為是腦袋有問題。只是，病態就是由此而生的。精神病患、瘋狂殺人、當自己的世界在一瞬間崩潰時，當倖存的自己在過去保護著自己城堡的殘骸中——車禍、裁員、政治鬥爭、飛機失事、AIDS、颱風，或是較不常見但驚人的地震——現在的我們比過去脆弱多了，任何人都可能由強烈的無助及茫然轉為瘋狂而不能自深淵中脫出，就因為幸福的假象被變動的世界在一瞬之間破壞了。而，這並非不可能。
所以對現代人來說，心理的建設及輔導相形重要。在輔導的同時，我們還不能忘了自己的立足點是多麼脆弱。

其他，這篇文章中還提到了展望。對於連明天可能發生什麼重大的事都不知道的人們，有的反覆思考著世界末日，甚至想得太嚴重，產生了飛碟教、末世理論等等；雖然我們一一踏過預言過的世界末日，就算是多麼權威的先哲、從沒出過一次錯誤的經典，在今日前都仍舊沒有言中過。看來這種事也不太可能一夜之間就發生，我們還是過一日算一日。不過，樂觀的來說，生涯規畫仍是重要的。生涯規畫——這也是近五年才活躍起來的新名詞。過去的人，人生的路幾乎是固定而沒大波折的，哪像今日如此多變，不作規畫就不知所措？因此，作者說了：「面對過去、走向未來」，還是只能走一步算一步。
為了之後人類的生存，環保是重要的。地球在人手的摧殘下能再住幾千幾百年是不知道，我們也只能盡量延長，希望最後一批住在地球的人們能說：「我們盡力了」。
2000.1.5-6

牽手成分手

來源文章選自張老師1998.11月號(251)P44-77，12月號(252)P106-121。

對於人，尤其是現代人來說，感情的支持是相當重要的因素。曾經有調查指出，結婚、離婚以及親人的逝世等人生中重大事件都會使人縮短性命。（參見社會心理學。此外人體內還有個基因使社會性疼痛反映到生理疼痛感知，說不定每次的別離都會造成類似的效果[補]。）但是人與人間越發疏離、隔閡越來越深的今日，感情的波動是否仍能造成如此大的影響，則尚無定論。一般來說，現在的年輕人比起上一代已不再那麼重視某些因素。分手，要分得乾脆。打擊，哭個兩三天重新站起來。沒有讓天掉下來的事，太陽打從西方出來也僅僅是個譬喻罷了。所有事物不會永遠不變，失去永恆的人們知道所有的堅持終有緩解的一日，而漸漸地不再重視許多傳統價值。
所以，現代人對分手也不再視作如此嚴重的事。依此次張老師的調查，過去（民國74年）考慮另一半最大的因素是家族，換句話說是傳統。過去人們重視感情的聯繫，所以男女朋友再怎麼也比不上活過二三十幾年的雙親（即兄弟姊妹）等。另一方面，也是因為不敢對傳統的價值做出反抗：假如父母說不，自己還私奔的話，多少會被說成不孝。不過，時代在變，價值也漸漸移轉。今日還有極力管束子女的父母，則反而只會招來不開明之名。此次的調查（民國87年），男女交往後分手最大的因素是沒感覺、不夠刺激。很顯然是個人化的因素。不再重視傳統價值觀（或是說被束縛）的我們，轉而以自己為中心，而冀求世界也繞著自己轉，假如對方不能滿足自己，就分手吧。第二個因素倒是一直沒變——第三者的介入。妒忌是人的天性之一，我的唯一有了另一個專情的對象因此導致分離，這當然是不可避免的。不過，換個角度想，在專情不再如此被重視的今日，還希望對方對自己專情的我們，也滿傷腦筋的。
另外一項是分手之後的感觸。在過去，人們多數冀望著能破鏡重圓，因為多少有些不捨。不過，現在調查的結果，是對過去的一段情抱持「遺憾」的情況多些。「遺憾」，但是不想再受到煎熬，所以頂多說說「你真的很好，可是……」罷了。（這就是現實的現代人）
現代臺灣人也還有處女情節，像是非處女比較不容易主動談分手；大概是以為「我本來就不是處女了，條件自然比較弱」吧。而分手後，男方大多有愧疚感，這是因現在的臺灣，女孩子和人發生關係，仍舊被公認為是吃癟的一方；是成本問題。不過，更多的是不把貞節當一回事的人。因此現代的交往多是肉體親密度高、但是交心的程度則淺得令衛道人士感到可悲。比起來，美國，甚至是日本都更加開放，其偏差（以過去的角度而言）更不在話下。只是，這正是現代的潮流，談不上道德不道德什麼的。
分手通常有主動及被動兩方。根據此次調查，主動的一方容易有愧疚感、遭受周遭責難「負心人」等，而被動的一方則容易覺得自我價值被否定、獨處很難過。交往一段時間後，雙方容易習慣於彼此依附，突然回歸獨自一人的生活，有時是相當大的打擊。鬱悶、心裡甚至是生理上的不適都有可能發生。由此看來，還是有很多人把心理上的支柱依靠於親友情人上。

總結說來，現在比起過去已開放許多。傳統若是包袱的話，我們已漸漸丟棄。不過，換個角度來說，卻只是我們的自制能力變弱，更加隨便而已。這樣的說法聽來令人有些痛心，不過，正因價值觀也沒有絕對的標準，對這些潮流趨勢，只能以漸漸轉變的日常觀之罷了。
