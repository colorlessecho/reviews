我乾脆來作個「R指定」名單好了（笑）。欸…上面有的還不少。自慰Ⅱ接近了。For You是典型、姉妹妻、義母一定會上榜的不用說。這一個也算是了，只要知道「忍者頭目的繼承者（女兒）幹上主角」，不是為 H 而玩這 game 的就可以準備走人了。（不過，只為 H 也不用落魄到玩這 game 吧，其他還有很多選擇啊。）啊，這麼一來，名單上的遊戲很可能會連破我「最短終結時間」的記錄。不行不行，寫心得的時間會比玩的時間長的 game 不能玩太多個（爆）。
這個遊戲感覺似乎有段時間了，系統的華麗度和天藍の夏差不多，原本因為想清清冰箱，安裝時嚇了一跳。「嘿？沒三秒鐘就裝完了？」這一定只有主程式及 REG，據經驗可以知道。所以，這 game 若不想傷光碟機，請虛擬光碟 drive。一打開發現 MEGAMI 字樣，心中就有不祥預感。『上次不知道是玩到哪個令我反胃的 game，沒記錯的話，上面也有 MEGAMI mark…這種公司也能在業界砌立不搖，世界真奇妙。』沒辦法，既然買了、裝了、那就玩吧。
比較特殊的是，在 bi-時加裝了「特殊效果」（上下左右搖晃），這在玩時就知道囉。其實也不算什麼，只是沒其他 game 這麼做就是了，令人「會心一笑」。故事直到最後一個選項都令我以為「直接把他打入 R指定名單就好了」，沒想到還有一點轉折；不過那也只是男人小小的羅曼，沒啥。看起來亦非初期之物，開始就莽撞想 H 的話直接 OVER。和記憶の海から～Watermark～一樣，save 不能隨時存相當傷人。啊，對了，這麼說來天藍の夏也是一樣的…（又多了個相似處）CG 普通，我不是很喜歡。
