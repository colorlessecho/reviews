讀書劄記 @ Colorless echo -the days never back-

家家酒
原本是當作 R指定來玩的，後來發現也不能算是普通学園物（笑）。和偽タクシー一樣，攻略の道沒發現其蹤跡。這種 game 通常是象徵「沒有保障」，好 game 不用說，就算古老攻略の道也應該有。
就算這樣，倒也不那麼差就是，只是要玩的還是算了，這劇本不是值得一玩再玩吟味良久的東西。解決方法？我發起狠把 save 檔全填 ff，結果就出來啦（笑），結局也沒很多個，幾乎全是理菜的。簡單說其他配角都不值一提。
分數是假設作家將此劇本好好發揮而打的，雖然光是看 end 以及頭十天我還看不出有啥扣人心弦的東西，姑且還是假設「他很棒」（笑）。這種分數的話，他起碼該把理菜留學後以及作各種選擇時的心理歷程都陳述出來，並表達「雖然愛著主角，但是因為…」種種心結。若非如此，只是照主角的選擇，漸漸就演變成那樣的話，那只能取較低的分數。我猜是後者吧，不然這 game 早出名了。
有 review，系統普通。不過選解析度不同時背景是用他們的圖填起來的；嘿，真可愛…的行為啊（爆）。音樂也是普通的樣子，沒去注意。CG，不怎樣。
