偽(ニセ)タクシー偽 taxi
恐怕很難找到比這更くそ的 game 了吧（汗），貨真價實的 R指定。從封面加上攻略の道沒這牌子的，我就知道不妙了。不過那也沒辦法，只是有些困惑，沒攻略要完全解決這 game 要多久。又不知道這 game 多古老了(1999)，用的還是 DirectX5。一開始的感觸是有語音但 BGM 處理太慢導致換 BGM 時延滯嚴重且無法關閉；後來發現，竟然還有駕駛計程車模式（爆）這在我玩過的 Hgame 中可是僅見啊（有點遊戲性。但卻不能玩警匪戰?!），而且有的 CG 竟然有4個 640×480 畫面，這也是這遊戲才有，雖然畫得普通。就算是這樣的 game，我竟然還是玩到了最後，也因為玩這麼久，才更感受到其無與倫比。還是改名くそタクシー好了(-_-)。
假如只是硬幹而不深究的話，一小時可以結束吧。之所以くそ，是因為他雖然是 R指定，不過是個「只要花時間就可以 clear」的 R指定；唯他會讓你覺得時間花得很沒價值。我總共也花五小時囉，對這 game，請務必使用他人的 save，起碼也利用一下攻略。更好的方法，真的要碰請看圖（雖然 CG 也已經夠差的了），不這麼作只是浪費您寶貴的人生罷了。話說回來，為何連這種 game 都能流傳到臺灣？那些商人還真是賣不擇品啊（恨）。
要用滑鼠或鍵盤可以設定，應該不用提醒了。系統真～的好慢。不知道是多媒體衝突啥的，每次換場面都要等上將近一分鐘。ED 不能 skip，沒有 skip，只有全螢幕，執行時又不能跳到其他程式，文字顯示慢，已經沒有這以上的くそ系統了吧。音樂？我實在不記得有哪首扣人心弦的了。
我能褒美這 game 的，大概也只剩下超大 CG 以及最後的啟示了吧，不過那東西 ELF 的 game 中真是一堆啊。
