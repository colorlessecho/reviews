讀書劄記 @ Colorless echo -the days never back-

一看到「雀」就應該有點警覺才是，美少女ゲーム界有「雀」的十之八九和麻將脫不了關係。這個找不到攻略的遊戲，事實上我也懶得找。結果，裝玩經過一晚上及一上午雖終於破了 story mode，卻仍有兩個人不知 CG 該從哪取得。free 對戰 mode 有兩個新面孔，但對打了好幾次仍不得 CG 之道。當然，對眼見不知從何下手的東西，既然不是那麼急迫，就放著了。
因為我第一顆 CDROM 是虛擬的，懶得換過來，而 MIDI 又不知怎的沒音效，所以玩時我並沒聽 BGM。不過麻將音樂反正就是那回事，一些親鬆滑稽的東西。後來(現在)聽覺得此部還不錯就是。總長一小時，雖然我不至於拿來單聽（水準還是差了一點點），不過比起普通遊戲的爛配樂是好太多了。系統有許多令我不滿處。skip 太慢，window 模式 load 實在太重，而 full screen 在跳脫再回去時螢幕全變黑的。加上 save 檔看似編過碼，（懶得試，比較小那個大概是 check code。）所以當場放棄，乖乖的和妹妹打麻將。只是，當和某位妹妹耗了二三十分，卻在對方作弊時當掉，（還連兩次！）這實在令人不太好受。
濡れ場中上，有不錯的語音。簡單的講，這是個麻將+濡れ場的遊戲，僅僅如此。至於麻將的部分，那令我回想起超過五年前父親在單色電腦上曾經玩過的東西，不知那裡出來的雙人麻將。麻將還是要四個人比較現實，所以只有雙人的遊戲我不太想玩。而 story mode 竟還有作弊，也能調難度，真是相當奇怪，令我不禁懷疑這該不會是傳說中不公正的麻將遊戲。也罷，總之過也過了。除了大王一戰外，我沒有「Magic」（作弊）過，事實上連打大王應該也不需要。只是可能會多耗些時間就是。出牌和 ELF 那款一樣，剛開始都是字牌比較多，這令我感到相當好奇，是不是他們的演算法或是亂數沒取好？現實中也會嗎？臺數算法及規則好怪，搞得我有些迷糊，曾集過一萬到六萬也只算一氣貫通。
玩到大王前我就有點悔恨這 game 不能改 RAM 了，看看即使全破我麻將功力也不見得會增加多少。終究還是硬著頭皮玩下去。CG 收集不用怕不全，因為是一直線，只要破了也就集全了。最後，我感謝上次 ELF 及此次幻想雀遊記給我的特訓，看來現在和電腦玩，我大概都有中下的程度了。下次特訓？再說吧（笑）。
