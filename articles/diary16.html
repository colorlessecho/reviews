早上搞完 ssl 的申請，到處查查，才發現 Gmail 邀請函變成一百封了。我通常一兩禮拜以上才會收一次 Gmail，上網一查，有的人八月二十五六號就已經增值了。然後發現現在 orkut 與 wallop 的邀請函似乎比較有價值，所以找了一下看有沒有好心的人士。閒晃著整理東西（看オタク文化作品）之際，很快的就收到了一個幾米寄的 wallop 邀請函。這東西簡單的說就是社群吧。或是說，Six Degrees of Separation 的實驗室？（對網路發燒友來說，甚至平均只剩下 4.74。）使用 .NET 動態產生 Flash（沒錯的話？），並結合了 blog 與更靈通的溝通功能。不過我想更重要的和 Gmail 一樣，還是他的觀念吧。這種東西主要是讓人際間的溝通更方便，有點像是 ICQ 或 qq 等等聊天軟體吧。不過，因為只需要用到幾乎每臺個人用電腦都會有的瀏覽器，而不必安裝軟體，所以更為方便。而且，藉由 server 端的一些特殊功能，還能看到自己的人際關係圖，這也滿有趣的。這些平臺的實現，或多或少表現了人與人間尋求溝通的渴望或是（起碼是被迫的）必要性吧。聽說 wallop 比 orkut 華麗，看到 Microsoft 那種以 flash 拚美工（？）的做法，我想的確不用多說。雖然滑鼠無法用右鍵，Shift+Insert 也貼不上（得用我記不起來的 Ctrl+V），但 wallop 的介面依然頗為方便，在我看來已經有點辦公桌的感覺了，可以便利貼。我只是 wonder 他們怎麼沒連 mail function 也作進去，讓 mail 看起來可以像簡訊。另外，現在看起來，最需要加強的是編輯功能。假如只是隨便貼一點東西上去的話，像 wallop 這樣只是一個輸入格是不錯。但若是有更多花樣要加，例如字型、顏色甚至圖片，我想直接在 blog 呈現區直接編輯，甚至直接將多媒體物件拖拉進去是比較好的處置法。現在能用的效果似乎很少，例如粗體只接受< b>，假如加上屬性就會被退件。其他方面，還發現有時文章太長時會被截斷。這可是重大疏失了。可惜，無論他如何好用，因為我惜字如金的習性，未來還是不打算搞本機之外的 post。

剛剛看完 Discovery「生命之光：40億年的進化」單元，提到生命演化歷程：生命自從四十億年前演化，三十五億年前使地球大氣充滿氧；五億年前物種大放異彩，現存幾乎所有最古物種都由那時生存至今（之前演化出最複雜的生物是水母），鱟等古代海蜘蛛遠親並嘗試利用陸上無天敵的特性（結果造成烏龜等在陸上產卵的習性，可惜今日這種習性已經得不到好處。），當時的陸地還光禿禿的；四億年前陸上植物繁盛，蜈蚣等昆蟲跟著也離開了水，造成至今以群居的螞蟻為首（數量最多），遍布大陸的情形；三億七千萬年前脊椎動物演化成兩棲類上岸；一路走來，進化（或者更應說是演化，對環境適應的驅力）迫使著生物不斷演變。（今天的節目也提供了我大學時的疑問：昆蟲與脊椎動物既是不同時期登陸，那各是何時、如何？當時教授雖感興趣，可惜這方面他也沒深入研究過。）我回想到 Punctuated equilibrium：生物的演化不是慢慢來的，而總是在某個時期突飛猛進。或許，應該改成這樣：演化和環境的壓力有關，例如當環境變得惡劣時，演化就會變得相當快速。但生物圈很容易達到平衡，所以在大多數情況下，演化總是沒那麼明顯。這樣的說法就能解釋為何我們看到的大規模演化現象有很多集中在某個時期了：因為，那些時間中，地球的環境遇到了突然的變異，使得生物若不改變就活不下去。
