SNS 考察

mixi の招待状、手に入れた。早速研究して貰いました。今回は mixi を含めて、全部六ヶ場所を比較するつもり。

この考察には大変失礼なとこがあります。間違いも一杯あると思います。心が込めませんから。もし気に障ったら、どうかお許しを。できるなら、教えてくれるとありがたいです。

前は全然目に入らないのに、何故今更SNSを気にして来るのだろう？それは多分私は手遅れに長けるから。最近ネットでゲームをする、友達を作るのは…盛り上げるとは言えないんだが、段々嫌でも気付くころだ。日本はネットで交友、韓国は両方で、台湾ではオンラインゲームが流行ってる。そんな時世で、ちょっとだけでも、考察は必要だと思う。
久しぶり Wallop を登録したいんだが、まさかリニューアルしたとは。ま、なんとか招待状を受け取ったが。それでいっぺん展開して、使用方法の違いを並べてみる。
SNS はそれぞれの特性があり、色々な仕組みがついている。以下、その辺りをちょっと紹介してみる。
自己紹介
写真、性別、趣味、特技、年齢など、これらは恐らく他人を認識するにはもっとも基本的な個人データ。言い換えれば、こんな情報があるとわたしたちは「この人はこんな人間だ」と思い込める。人と関わり合うことに相手の特別性を確認しなければなりませんから。写真、自己紹介などだけではちょっと頼りないかもしれないんだが、まあ、現実での向き合い会話もそう頼りできるものじゃないね。ボディーランゲージを読めるなら別だが。
Cyworld ではちょっと変わりました。写真、性別以外に幾つの問題を用意して、それを通じて紹介する。例えば「今の生活と未来の夢」、「Cyworld に一番見せたいとこ（注目エントリー）」、「皆に話したいこと」等等。
コミュニティ
これはもしかしたら、普通の SNS に一番盛り上がる場所。面白いことに、Yahoo! 360°にはこんな機能が備えていない。Yahoo! 360°は「My Page」と「My Blog」がメーンかな。Ctworld なら元は主に「迷你窩」だが、最近もとうとう同じ機能の「秘密基地」を増設した。
メッセージ
個人的な話し合う方法。掲示板ではP.M.を使うのもある、つまりお互いのやり取りだね。友達が多すぎると大変になります、返事が間に合えません。簡単に言えば、普通携帯でメールを送る高校生大勢いるんだろう？ま、そんな調子で。
そう言うなら、もしかするとメッセージこそ SNS の最も盛り上がるサービスじゃないの？どの SNS もこの（或いは似てる）機能があるし。
脇に新聞を並べ
これは多分 SNS を唯一の門にする方々のために準備した service。どうやら mixi とネットシティ以外はこんな仕様ないんだ。便利と言っても、ロードが重くなる。
あしあと
Wallop や Yahoo! 360°とならべ、日本の SNS は一つ面白い機能を付けられた。それは「あしあと」です。orkut では「最近閲覧したユーザー」と言う、同じ意味のものがあるんだが、それはあくまでも Google Talk 関わり、通知のようなもの。Ctworld なら、「最近一週內」と言う記録があるんだが、字面通り一週間だけ。mixi やネットシティなど日本国内の SNS ほど履歴を完備的に記録するには敵いませんね。日本の使用者にはこの機能をどんなに必要になるのか、ここでその重要性を示すのだ。
しかしどうして？私の考えは、日本の使用者にとって、人との係わりはかなり大切だから、もし人ん家を訪ねて、話とかなにも残さないほうは随分無礼者だとでも思うのかな。だから、足跡を返すものにもおかしく思いません。
それに、これは日本だけでなく、ここでも同じだが、現代人は忙しすぎるのだ。朝早起き準備して、仕事に出て、夜遅く帰り。職場でも上司下屬の関係を維持するのはもう精一杯だ。他の人の事情に深入りするのは、言わなくても、ごめん。だから、一日が終わる、静くなった時、もうネットしか話し相手ないものも大勢いるじゃないかな。だから足跡のような小さな（他人と通う）きっかけでもうれしく思えます。
来訪者をもっと知りたい願望は皆ある、だから色んな来訪データの解析ツールがある。例えば Google Analytics。勿論足跡もその中の一つに過ぎない。できるだけ詳しいのがいい、個人的のさえも大丈夫。だから誰が訪ねたのか教えてくれるとありがたいその一心だ。まあ、そこまで酷いでもないんだが。
年齢、趣味などの条件付きで人間を検索
どの SNS もこの機能は備えてる。まあ、友達を作るサービスだったら、そりゃ仕方ないと思うよ。でも、どうしても人間をモノと同じレベルで見なされた感じがするだな。人間はそう、もっとガラスのようにデリケートなもんじゃないか？石扱いは酷いじゃないの？（いや、違うし。）
以下、各 SNS の特殊点についてちょっと話します。
mixi
日本最大手。招待制だからそう簡単に入れないんだが、そのためにメンバーの結束力が強いじゃないの？交流に特化する機能がある、例えば足あと。Firefox で、絵文字は使えない。 不足もあると思う。時々ログインしている人や友を知りたいのに、どうしてもそのような情報はない。それにチャットがあると一部の方にとってもっと便利になる。
ネットシティ
人数、少ないね。こう言うのはちょっと失礼かも知れませんが、mixi に倣っているの？動画の service はないんだが、プレミアムが付いてない、これは mixi より優れる場所。
Ctworld TW
なんでも文字じゃなく、絵で表示。でも絵の下手糞はもう一つの統合掲示板インターネットコミュニティ、巴哈姆特を思い出せる。
IE だけかはしらないが、「我的房間」は Firefox では使えない。
orkut
主にコミュニティ(group)機能だと思う。
Wallop
リニューアル前と並べ、早くなったような気がする。Interface が綺麗。右クリックが使えるもの利点。
Yahoo! 360°
主に blog かな？
これからまた大勢な人が SNS を利用しにくるだと思う。携帯のように、皆も SNS を通って他の人と話し合うことになるのだろう。その時、声の service は文字や絵や音楽や映像は同じ service で扱えるだと思う。
それで。何か思い出したなら、また書くよ。

高校の時、私は毎日欠かさずに日記を書いていた。大まか一日千字から二千字くらい。それで曾子の「日三省吾身」を達成したと思っていた。（本当はそんなことはなかったが、それでも反省することはできたんだ。）そのせいなのか、私は自分の欠点が見えなくなったと思うばかりに、学友に問う時があった。（多分、慢心とは違う。どちらかと言うとむしろ謙遜の方。ただもっとはっきり言うと、無知としか言いようがないね。悪意がないとは言え、時には無知でも人を傷付くことがある。）勿論、真っ当な答えは得なかった。

欠点って、果たして何なんだろう。人に求めてやっとわかるものでしょうか。違うね。年を重ねた今だからやっと分かること、人にこんなことを尋ねれば、その人が自分のどこが気に入らないか、何かおかしなとこがあるか、そんなことだけが聞ける。それは欠点ではない、人によって答えが違うもの。答えができるのならそれはそれでまだいいんだが。普通の場合はほぼ仮面を通じて、偽りのものばかり。逆に変人と思われる。そうだった、当時の私はただ非難されない状態であって欲しかっただけ。浅はかな考えだな。
そう言えば、あの時の私はまだ唯一無二の真実（価値観）を信じていた。今はもうできない。人生的意義にも言ったように、全ての時代及び全ての人間において「絶対に正しい」と言うことは、多分ないだろう。あるのは各人の選択のみ。だから他人（の行為など）に口を出せるのも、その人の価値観自身に矛盾が存在している、或いは現実を上手く解釈できない、そして周りの人に受け入れられなかったときだけ。
今の私に、「欠点」はそのように現実を丸く解釈できない、或いは周りの人に受け入れないと言うときに使う言葉。だから、もし条件を満たせるのなら、たとえ自分が気に入らなくても尊重するべきだと思う。一つ注意すべきことは、「尊重」を求めること、それは「公正」を強要するのではない。(2007/10/29)この世界に不平等なことは多すぎた、物理的にも、人間性においてても。人と人は違う基準で立っている。しかし同一な標準が望めなくても、せめて尊重することができる。「正しい」という価値は存在するかしないかさえ知らない私にとって、それではもう精一杯だ。
