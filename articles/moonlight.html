蒼月,蒼茫月色
老實說，雖然一直以為 Studio e.go! 就是那種專靠 CG 獨挑大梁的傢伙，但看看此部，他們多少也有些進步嘛（笑）。該不是山本阿姨錢賺夠多了，可以請好的作家、作曲，以錢養錢？（爆…此句帶刺，還是別亂說。最起碼山本阿姨應該還是畫得很辛苦…還是培養出人才幫他畫了？）總之，這部在我評價的重點項目中每項都有不能忽視的實力。
畫，不用說；只是我到現在才發覺，為何眉毛下的影子那麼明顯？只是一瞥而過還不會覺得有啥不對，但仔細盯久了，那就變得很奇怪。每個人都那樣，尤其某些人物像，讓我覺得那簡直就是外星人（爆）。還有，人臉的造型只有三種，男孩、女孩及男壞蛋。各代主角都是男孩，幾乎每個女的都是女孩造型。仔細看，他們的臉總是盾牌型三轉折，猶記國高中曾有段時間想學學漫畫——起碼能有基礎程度（當然是無疾而終），當時一開始參考的是宮﨑駿的天空の城不說（這部動畫當時重複看了三四次吧），後來我畫的人物臉也全變成盾牌型了（笑）。不只是他，日本流行美形的那年代，天地無用！及一大拖拉庫漫畫也全是這種造形的。看久了還是覺得有些好笑。第三種，在エレンシア戰記中也有出現，此部中當然就是人気最末的アノン了。那…簡直是一模一樣，沒看到啥改變。對了，說其他也不是沒有，エレンシア戰記中那老頭吧，還有，硬要扯的話…大概就是每部一定要出現的雀（？）了。有的（大多數？）雀是陪襯，有的比主角更與女孩親暱，與主角的關係卻不怎麼好，性格還相當彆扭，死小孩的雀版：死肥雀。他們乾脆把公司標誌用跳動的デボすずめ代替不就好了？大概是山本阿姨用女王的態度及漫畫中常會出現的那種陰暗的臉，（抱著搖著嗷嗷待哺的…）再配上星狀閃耀的眼威脅每任的劇本：「你也知道，我每部作品一定都會出現這隻すずめ，你不加給我試試看！」吧？所以，儘管這部劇本中沒出現那隻雀，阿姨還是親自指示在圍裙中畫上了隻大雀。（…也許這已成 Studio e.go! staff 共識，不畫他就不舒服？←爆²）好吧，無論如何，Studio e.go! 的作品畫一向很精緻，精緻到令人愛憐；也因此會有那麼多人收藏：正如有人一聽到「這歌是 I've 唱的」就會馬上下手一般，聽到「這是山本…」就當場掏腰包的也不少（笑）。
系統，該有的都有了。不過 color set 設完後竟然會自己亂跳。還好，エレンシア戰記那時沒作用的即時顯示在此部中有進步。只是，還是有些問題，好像在 My fair Angel 才比較正常。Ctrl skip 到未讀過的停止，直到 My fair Angel 都還沒作用，可能要等下一版了吧，暫時用 F3 可以替代。這些舊的 bug 恐怕不用期待他們出 patch 了，只是在新的系統中會修好就是。full screen 跳出再回來時會在下一句臺詞回復正常，在 My fair Angel 則在跳回來時就已重繪了，可以見到系統的演進喔（笑）。另外就是 skip 還是要等很久，這在劇情量多的此部中更是一大不便，skip 好～久～(~_~)。語音，エレンシア戰記沒有的在聖魔大戰リニューアル起也加上了。嗯，有進步。（嘿，我幫你們測試系統，不用感謝我嗎？←爆）
音樂也是有進步，可惜水準…仍有相當加強空間（一句）。#(3, 5,) 6, (7,) 8, 11, (12, 13, ) 15–18 是比較耐聽的。最後就是最重要的劇本了。……還是覺得中心主題有點老套。以下（ネタバレ!!）為個人低見，或許有些小瑕疵，如專有名詞用錯，請別介意，大家知道是怎麼回事就好，因為我也不是專門研究宗教（如佛教）的人。當然，能提供建議，批評指證我錯誤的，我將非常高興。佛教之所以吃素，是因為視萬物為有情眾生，（專有名詞用錯？總之，就是萬物都具有某種程度的知性及感情，如同人一般。）而起憐憫心，（不如說是同理心，因為他們不認同強者為王，害怕哪天自己變成弱者會受到相同待遇；或者甚至在生活的小細節中也是一樣。他們和絕大部分人一樣，喜歡公平公正，以及統一的系統和統一的世界。）也為減少業而盡量不食肉類。在他們的世界，生物也是有階層的，人屬於相當相當高的一種。此部中心主題是「悲しみは、絆から生まれる。（悲傷由牽絆而生）」而防止絆的斷裂或增加新的絆（這簡直就是轉移注意力-_-）可以減少悲傷。這應該是西出明久人生觀的一部分吧，作品通常會透露出作者部分的人格。不知和ONE有沒關係，ONE 中主角能留在人世就因為和人世還有牽扯。當然，還有個方法就是都沒有絆，也就無從悲傷。佛教採取的是這種方法，心無旁騖，自可專念成佛。（這麼說 ONE 的主角…不，人家佛不是這麼成的。←笑）簡單的說，絆也是業的一種，還是少造點會比較好。也許他以為在人世中就不可避免絆的存在，所以無論如何也希望能活的快樂些。只是，防止絆的斷裂（促使主角不對普通人下手，並改為制裁アノン。）還能理解，很多人也是這麼想的，不過增加新的絆，造成壞苦就有點矯枉過正了。如此一來一方面增加絆斷裂的機會，另一方面這只是用新的假象將過去傷痕掩埋。但傷痕是不會消失的，他一直存在，等待著哪天再度被 reveal。根治，應該會比較好，理解是有其力量的。何況絆不是那麼簡單，能以加加減減處理的東西。（與母親絆的斷裂，可以用新加入與主角的絆中和?!）是故，我並不完全贊同主角將女孩收留自己身邊的方法。
中心主題處理到這，似乎盡是批評？不過，故事的營造能力，還是不得不說是有其程度的，有幾個結局不錯。主角此次的贖罪大概到小的女孩嫁人告一段落，而在兩位都辭世後終止吧，本來無盡的生命就不該作這種事的，徒增麻煩以及自己與養女的悲傷罷了。（也罷，假如算是增加一次經驗的話。）原來的設定也真老套啊，「高潮時的精力最營養」，不知被用過多少次了，尤其是アダルトゲーム界（笑：好用啊）。總之，此片還滿值得買的。
