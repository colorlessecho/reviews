在语义网（Web 3.0）如何整合应用程序（原文Integrating Applications on the Semantic Web）提到未來我們應該能夠整合資訊，讓程式也能「閱讀」我們寫的文章，並自動幫我們作資訊統整的工作。說的也是。Web 2.0 時代的精神似乎（因為我實際上並不確定）是以網民（更準確的說是有發言權的個人）為重。（這讓我想起以 You 作為「風雲人物」封面的 2006 時代雜誌。我晚人家兩年了。）所謂 UGC，人人可發表，大家都能參與，或是 SNS 都是這精神的延伸。而今文章中的技術已經具備，需求也漸漸展現出來，相信往後會出現相關的應用吧，例如數據共享之類。只是這篇是五年前的文章，而我們如今還沒完成，這點令人有些感慨。另外，或許人人可參與也不代表平等：參與多的人較有發言權，自古以來皆如此，所以才有 Digg，維基百科以及 Web 2.0民主的迷思（原文 Digg, Wikipedia, and the myth of Web 2.0 democracy.）提到「在 Digg 和維基百科，少部分用戶 (power user / power group) 擁有特別大的權威。」與「維基百科 administrators 承擔了接近一半的編輯任務。」事實上 Facebook 也有類似的密集性勞力現象。或許，所謂 Web 2.0 就是可能讓少數某些活躍的參與者有「家」的感覺，然後自欺欺人說「每個人都參與了」，其實貢獻分配非常不平均。但換個角度，若是能抓住這個重點，營利機構將有辦法賺到更多錢。後面還寫了篇媒體對個人生活的影響，也提到這些東西。
近來網路上在需要徵求使用者同意時開始有種趨勢，會在作了之後再顯示一個「undo」的 link 或按鈕等。只有在作重大決定時，才會跳出 prompt。這樣可以讓使用者不需中斷連續的工作去應付提示，並且能達到誤操作時也能回復的效果，是比較理想的方法。不過，若能像普通 editor 一樣提供更多的 undo 與 redo，甚至選擇要回復到哪個階段，我想會是更理想的介面。

この文章を見て、気に食わない！と言いたい方もいると思いますので、そうなりたくないと思う方は避ける方がお勧めます。

前にも言ったように、…いや、ここでではありません。元々載せるつもりだったんですが、事情が事情でチャンスを逃しました。今更補足しても無意味だと思いますが…
先月末、通勤途中に事故に遭いました。その時は道に倒れ、中々動くことができませんでしたが、救急車に近くの病院に運ばれた後、三日も過ごさない内に退院しました。私がずっと「すみません」という言葉を繰り返したせいか（そもそも怪我したのはこちらでしたし）、相手は優しい態度をとっていました。丸々一週間も過ごしたが、今はまだ仕事を再開するのはまだ無理だと思いつつ、自宅でのんびり寝て喰って過ごしていますが。でも、ずっとそうしているのは申し訳ないから、来週復帰予定です。
今回の事は（中国語で）別途に記載いたしますので、ここではスキップさせて頂きます。結論から言うと、事故に遭ったことより、自分の無感動さによっぽど驚きました。呆れるほど。こんなことに遭い五体満足に動けない時、「嗚呼、人間はなんて脆弱なものだろう」とか、「ああ、だから身体障害者は我侭なのではなく、そうしなくてはいられない苦衷があるのかな」などと思うのが一般常識かもしれません。でも私はちっともそんなことを思い浮かべませんでした。ただ周りに起こることや、時の流れを見過ごしていただけです。そう言う人間的な共感が欠けています。元々駄目な人間でしたね、私は。（苦笑）

何も感じないなどありえない！と言われたら、確かに全然無いとは言えませんね。強いて言えば、一番感慨深いのは多分疎外感でしょう。私の場合、一人暮らしが長いため、孤独こそは感じませんが、それでも時に人間の矛盾さに戸惑うことがあります。今回の事故で同僚が沢山お見舞いに来てくれました。普通の状況でも多分親族や多くの友人が来るのだろう。でも私は何時も考えすぎます。時が過ぎると皆自分の仕事があるから毎日来るのは無理でしょう。一年、二年、もし数年も経ったら自分の事を忘れてしまう人もいるでしょう。そして自分の存在はどれ程ちっぽけなのか思い知らされます。しかし人は必要とされていたい、他の人と触れ合いたいのです。人間は殆どそうありたいのに、そうなりえません。なぜなら、やはり人は元々独りですから。
こんな苦しい立場を抜け出す方法はかなり簡単なのです。いいえ、必要とされている人間になるのもいいのだが、ちょっぴり私の趣味に合いませんね。それは他人にとって自分がまだやるべき、或いは自分にやらせたいことがまだ残っている、と言う不完全な状態です。世界は自分を中心に回っていないから、「私は忘れ去れつつある存在」とは思わないで、代わりに「どうしたらあの人たちの助力になれるのか？」と考えるのはどうでしょう。人間は社会的動物ですが、社会はただ一人の欲望を満たすことだけで成り立つものではありません。他の人の記憶に残るより、もっと有意義な行動を取るほうがいいではありませんか。だから、たとえ必要とされていなくても、もし他の人が自分のしたことでもっといい生活を過ごせるのなら、それはそれでいいではないでしょうか。つまり、正常の人間でありたいのなら、たとえ自分が入院していても、自分のことばかりで悩むより他人の事を考える方が大切だと思います。
でもね、どんなに偉そうなことを言っても、自分から踏み出さねば意味はありませんよね。…いや、私のことではないですよ。身を投げ出すのは苦手で、どちらかといえば人と水入らずで付き合える器用な人ではありません。そもそも健康で正常な人間でもありません。まあ、言い換えると、私はわがままでそんな選択肢を選びたくないだけです。そんな言い方もできますね。

バイクで昼飯を食べに行ったとき、一寸気が抜くと踏み外し、倒れかけました。バイクは 90cc で古くて軽い方でしたが、それでもあの瞬間、腰が抜けかけ…いや、ちょっと痛いだけです。最後はなんとか立ち上がりました。しかし、数日の休養をとっただけで、回復した自信があったのに、どうやらあれは過信にすぎなかったようです。

お見舞いの言葉は必要ありませんよ。こんな失格人間の駄文を見るだけでありがたいのです。もし気に障ったところがあったら、気にせずスルーしましょうね。どうせ私のオナニー文ですから。
でも、もし何かを感じた、或いはこの不才を正しく導きたい方があるのなら、どうぞコメントをください。その方はずっとずっとありがたく思えます。

這種東西總令我想起之前有名的愛之味健康の油切綠茶（那介紹中的顏色看起來像是烏龍茶或普洱茶）等的油切茶事件，之前那個功效被誇大了（不具療效，不太能減肥。），但這次的聽說確實有不錯的效果。一早九點睡醒空腹（聽說綠對胃不好，還是別這麼搞。）喝下五分鐘就開始跑廁所。排便的感覺有點像拉肚子。兩個多小時後算是結束了，整個下來上了三次，體重前後消去 1 kg。有人說會多排氣，我沒這現象，倒是之前發生車禍的脊椎部位痛了起來。不知道其運作原理是不是阻礙脂肪的吸收？在衛生署核發健康食品許可證一覽表有列出資料摘要。本產品標榜的是含有菊苣纖維與兒茶素。後者在平常喝的新鮮綠茶中更多，所以就不多提了。菊苣纖維有人說與跟「菊芋纖維」是一樣的東西。正式名稱好像是「菊糖」，不過若是用「糖」這字來廣告，肯定沒幾個人會為了減肥買吧。主要效果看來正是因為這東西而起。另外，雖說「甜度是普通蔗糖的200到300倍」，但這茶卻一點也不甜。我問了一下，前輩說很難喝。「很難喝為什麼還要買呢？還一整箱的買。」很顯然是健康因素。我覺得的確不是很好喝，但也算普通了。或許因為平常喝產地 TWD2000 一斤的茶（算中階），所以對這比較沒感覺。聽說前輩以茶代水的都是產地 TWD3000 以上的（特別強調「產地」是因為透過經銷商或中小盤商會貴很多。前輩是喝茶大戶，都直接殺到茶農那討價的。），那他們以平常所喝綠茶的標準來審視這個產品，當然就沒那麼能接受了吧。以前輩為例，現代人為了健康，就算不好喝的東西也能欣然下肚，甚至犧牲享樂來換取健康。只是，不得不為人的這種現金な態度感到敬佩。假如不是能減肥，能夠受到他人更高的評價，誰會喝味道不佳的飲料呢？更甭提要大賣了。這麼說來，我喝綠茶也是因為同樣的原因。事實上我不覺得茶好喝，只是取其讓人清醒的效果以及維生素罷了。抗癌或防心臟疾病啥的基本上不期待（應該說不在乎）。近來物價上漲，看起來錢變小也有相同的效果。很多人改用單車或步行上班以「省能」，假如不是比較省錢，有誰會這麼做呢？說的越多，卻越是哭笑不得。人求取自身的尊貴，卻無法擺脫實利的糾纏。我們在這種狹縫中求（心理層面的）成長，也真是辛苦了。（啊、酸味好重…）

全民記者

現代資訊科技進步，演變到前幾年出現了 Web 2.0，個人媒體時代來臨，且餘波盪漾，至今依舊火熱。如何讓你在幾小時內丟掉工作提到「微博客服務 Twitter 越來越引人注目了，突發新聞的報導，尋求問題的答案，它的時效性是 Google 無法比擬的。」2008年互聯網超越報紙成為美國第二重要新聞媒體。而網路對即使是大陸這樣的國家，其影響力也漸漸攀升。（注：上列連結有為搜狐廣告之嫌）第一次看到全民記者這詞大概是一兩年前；當時的記憶是因為科技發達，手機也能發布簡訊到 Twitter 之類的 micro-blogging service 上。（這邊所提的並不限於 Twitter，畢竟有許多人不過是為了趕時髦才用，和 Wikipedia 一樣大多數內容由小部分用戶產生 ，而其忠誠度也不見得高。）許多新聞素材在這些即時性網站上都找得到，這造成了突發事件最新的消息與狀態，不是顯現在電視上，更不是在急忙趕到現場的記者或號外中；而是出於事發當時身處現場，第一時間以隨身終端發報的民眾手裡。最快得到消息的，則是瀏覽這些網站的民眾。

OhmyNews

這幾天搜尋時意外發現韓國 OhmyNews 從 1999/12 就開始營運，是世界上相當早的例子，其經營模式或所造成的現象等也為許多英美研究者所提及（如 Online news, p.130）。TBS 於 2007 CE 推出 PeoPo 公民新聞前甚至還實地查訪過 OhmyNews，以習得其經驗。
OhmyNews：一場打破專業壟斷的公民運動提到「新聞記者比一個通訊員好到哪裡去了」。過去新聞是專業。其原因之一是發聲管道就那麼幾個。（辦報、建電視臺多花錢啊！）如今資訊環境成熟，人人可發聲時狀況就不一樣了。固然專業寫手可能有其對事件背景的了解以及筆法熟練的優勢，但人很容易坐井觀天。條件允許每個人都發表意見時，自然是這麼做較好。OhmyNews 的口號為「每一位市民都是記者」，並使用名稱「新聞游擊隊」來稱呼這些自願者；顧名思義，就是追擊新聞的人們。OhMyNews 經驗看臺灣獨立媒體發展一文提到，OhMyNews 打破了韓國傳統媒體「路線記者聯誼會」共生結構，改變傳統記者貪腐文化。OhmyNews 於其發刊辭提到，記者是擁有新消息、想要把它轉述給其他人知道的所有的人。但媒體將自己的發聲管道視為特權，（讓我們思考「無冕王」、筆利於劍的意義。）過大的權力導致自身的腐敗。臺灣現在除了人民素養不夠高、媒體沒有矜持而常隨群眾喜好起舞或各有各的偏頗立場之外，藍的藍、綠的綠，根本無心公正也是很大的問題。相對於監督政府的媒體與輿論，「全民記者」相當於媒體的監督人，做的好可以起到批判媒體的作用，進而達到媒體與社會責任中，從各種不同角度來思考，進而達到圓融價值觀的理想。

除了發表之外，連新聞的重要性也可以交由大眾評判。Media 2.0 全民記者 人人專欄 民主投票 決定排行提到 Digg 就是個具備這樣機制的網站。雖然這得要考量到交付權眾不一定真的能得到我們所預期、平均性的結果，因為不會每個人都擁有同樣（或我們所期望權重）的發言權，也不會每個人都參與其中（反而常常由少數 power user 掌權）。此外我覺得在臺灣，重大議題交付民眾票選可能不是好主意；但其他不常涉入個人價值觀的事件，只要參與者夠具有代表性（能使結果反映出事件真正的價值），這倒不失為可以考慮的選項。
總結從媒體與社會責任到上面提到 OhmyNews 的理念，因為我們可能沒有很多時間接觸所有的新聞，因此像 Digg 一樣交由大眾選出重要的新聞，我們不必費太多不必要的時間閱讀不太需要知道的新知，也可以藉由評論廣泛知道眾人的想法，這應該是未來可以嘗試的方向。但是這些嘗試可能在不同國情或民情下皆需有不同的修正。
