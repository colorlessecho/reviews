Unbalance（アンバランス）
一開始又是死?!那段英文的感覺不錯，可惜真的只有那麼一小段；那段也令我想起Portrait。當然，純以シナリオ來說，此部還不足與 Portrait 相提。大致說來，這 game 感覺相當不錯。最主要歸功於設定吧，主角的那種沒有感情，無法引出同理心的異常。可惜這方面沒有多加著墨，否則應能發揮到和 ONE 相當的摧殘程度。實作方面，大概得安排特殊女主角，作一兩個 True end。不論是鈴原さやか，神谷美香或藤宮律子，三個女主角都令我有描述不足的遺憾。不過，這是因為我「期待」更好的緣故。比起普通的 game，這樣已經算不錯的了。
音樂，一開始得那首不錯，雖曲目不多，但光那幾首就可以撐撐場面了。具體說來，就是 Theme of Unbalance, LOVE, Misato's Memories（#11, 9, 10）。這幾首的出場，不是解釋重要劇情，就是深層心理描寫；每次 H 完馬上跳出這些片段時我總會突然冷下來，漸漸發現雖然濡れ場很多，但是那都不是重點（笑）。配音也配得相當不錯，尤其是那些配角，女服務生及黑道混混之類的（爆）。女服務生有活力的聲音好像在哪聽過啊…進藤？那稍帶有爆笑意味的元氣聲音真不錯啊。最近玩的幾個都相當有水準，讓我幾乎要誤以為之前那些找歐巴桑來配妙齡的 game 其實沒存在過（爆），而現在的 game 配音注重腳色性格、情境氣氛的配合；這是理所當然的。還有那效果音ぷちゅぷちゅ的，非常值得一提（爆）。
畫的那不知道是哪些人，從永遠の都（第一作）起那些，感覺都不錯；這點就甭提了，我基本上沒啥美術細胞，評好評壞說出來也不具有公信力（笑）。就算如此，那舌頭畫得真好啊，之前的 game 都不知道在幹麼，應該多學習學習了吧（爆）。
系統在最近的遊戲中算中等，並支援鍵盤。不過，save 有時不會照所希望的作，這真是相當麻煩；公司尚未出 patch 的樣子。Hint 滿載，加上劇情簡單，基本上不需攻略。可惜圖片的處理稍慢，skip 時可以感受得到。
一邊玩我就疑惑著，到底哪種伴侶對主角來說才真的是好的？七個結局，六個不是 H 就是得不到解脫。美香 Happy END 的解釋是「彼女是不一樣（類型）的人」，不過，不管是哪一個，女方最後總是處於被動、附屬、奉獻的狀態，不，他要的不是這樣。他要的不是純粹接受，還需要一種對等的態勢才行。所以，美里才令他懷念。不只是因為長期的交往成習慣，也因為美里能適時對他提出意見及批評，而非一味依附他。所以，這部的結局在我不盡滿意。設置個 True END 是必要的吧。
