讀書劄記 @ Colorless echo -the days never back-

地獄的幽靈
這個，更接近電影了！雖然還是有許多缺點，老套的劇本等等（クライング フリーマン+静かなるドン+MONSTER+…？）：不過若是所有 game 都能有起碼這樣的程度（要求過高了吧），那就沒什麼好抱怨了。相對之下，一些片子數極高的 game 和這樣光一片就能滿足人的比起來還真不知在幹什麼。
system 的確像是自己作的新東西。說類型嘛…比較像終ノ空吧。可惜 keyboard 幾乎全不支援（唯一一個還是くそ的）。光鍵盤在那像是擺好看的，滑鼠則連珠子都拔起來了，只祈求速度快一點，使用上手的姿勢更自然（右手肌腱似乎已經拉傷了）。滑鼠游標能隱藏自然是最好…這部分和光を…是完全的對比。CG 部分以及遊戲進行時畫面交替速度太慢，不小心按到 Esc 馬上容赦しねぇ的跳出，改過的設定不會儲存…還是有相當多需要改進啊。
CG，傳承好 game 的固有傳統（爆），真是差到吐血啦。我真的開始懷疑，是差 CG 造就好劇本呢，或是好劇本產生差 CG。不過，大多數說是差 CG，其實卻是くそ型的差（換句話說就是「很有特色」吧(汗)）。為何會這樣呢。
說劇情…「この胸の中生きて…」最終完成版？！總之，この胸の中生きて…最後目標應該就是這樣的劇情了吧。接近無情的人生，卻又像作家在搬弄文字（竊笑）。
故事中，已選擇的總有無法回頭的感覺。抉擇只有一次，錯失了就無法挽回了。而即使如此，還是有很多部分重複——就像真實人生一樣，即使改變一丁點，大部分還是以相同形式流逝，無力回天。故事中沒有大團圓的結局，有的只是空留餘恨或是雖然幸福入手，總還有幾分遺憾。好像總要犧牲幾個人似的。劇本編排比較像是電影及美國風（？？），（還有蒙太奇手法，以及其他許許多多…吧。）而稍跳脫了傳統的美少女ゲーム。花了心以求前後協調，並且每個 end（全是 bad 的）幾乎都作了相當長的完整敘述，使得這 game 的每個結局都值得一探，不若有些 game 光主要的看完就差不多了。更難得的是 Phantom 將各個腳色融合在一條流上，不像 Refrain Blue 或 KANON, AIR 等分腳色撰寫。這種情況下要整合各部分腳色結局相對來說是比較困難的。（我一直不敢將ビジュアルノベル劇本和諾貝爾獎等級相比較，所以只好這麼說。）這些，都是值得嘉許的。不過，為何最後每個人總是想到男廁電人及化學室爆炸啊？同樣的點子看這麼多次，skip 很辛苦啊（爆）。最後エレン的 H 有點牽強。
褒了這麼久，還是有缺點的。同樣是人殺し屋的悲哀，此作煽情度就比不上銀色第一話。音樂，更是不用說了。雖然已經不錯，但真希望此作能有銀色的程度啊，那種值得拿來當 BGM 聽的。所以說，這 game 只有 ONE 的相同 level 吧。
總之，這 game 的突出處還是在其劇本。不用說，值得一買。即使是在兩天中迅速解決九個劇本，我似乎還是多少產生了失落感啊。
