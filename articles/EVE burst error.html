讀書劄記 @ Colorless echo -the days never back-

比起 EVE TFA 甚至 EVE ZERO 的系統，這舊型的不能用 Ctrl skip，聲音還需要滑鼠右鍵或 Esc 才能跳過，還沒有 review，是不便了些。CG 及 movie 也是古風，還有「全是 one cut movie」的評價。事實上也沒錯，那時 movie 沒技術作那麼精緻。
承襲…不，這是始祖吧，那搞笑的手段隱約可見，例如食堂中全坐滿 EVA 的腳色…果然，EVE 和 EVA 是有關係的，在 EVE ZERO 中甚至連讀音都一樣！（笑）
至於故事，なるほど。這的確是個事件安排縝密，而說到（催人熱淚的？）感人場面，最後的真彌子部分又相當不錯。EVE 系列到此應該可算「該玩的都玩過了」，那種傳說中浮不出臺面的 LOST ONE 除外的話。關於日本出來的劇本，相較於西方的，長久以來一直令我有「較富情感」的感覺，說得負面點，就是「濫情」吧，不過我本身好像滿喜歡這種寫法。很遺憾的，即使在大陸的作品，我所看過的區區幾本小說中都沒出現過那種表現法。這應該是因為國情以及日本長久以來的寫作傳統所致，連看 H 小說都能夠感受得到民族性的不同。所以，雖然光以偵探劇本來說，此作可能排不上頂尖，不過讀起來確實令我相當滿足，尤其是以美少女ゲーム界之內，此作說是偵探劇本五隻手指頭能列出的名次，當之無愧。以劇本事件的縝密性與真彌子那催人熱淚的最後告白，更是至今玩過 EVE 系列最高之作。只可惜當時製作時，雖然可能考慮到會出個前話，不過在接後話時就有些出入了。所以 EVE 系列光是靠 game 本身尚有邏輯上的瑕疵，得靠玩家自己用想像力更正劇本補完。
EVE 系列普遍不如其他以劇本為重的頂尖ビジュアルノベル那麼具有文學性，不過卻以其事件安排另有番春秋；只是，以文學性來說，總有此作為此系列表現最好的感覺。那就不多提了。以邏輯來說，雖然沒大失誤，但從四樓跳到游泳池，多少會無法知覺個幾分鐘，還有私人游泳池通常不會有足夠深度…首相明明是「即死」，還有時間寫「d」嗎？另外是有趣的 hack 橋段，在 TFA 中也出現過，劍乃好像很喜歡搞這套。那什麼系統啊？密碼還會顯示在螢幕上。hack 方法應該是偽造 IP 封包吧，不過在第一次使用的他人電腦上就能幹到這程度，冰室想必平時就在搞這行的，而且有一整套熟用的工具才是。就算這樣，中途出現能夠自己選擇項目閱讀的橋段也不太合理啊。還有輸入方法，相當奇怪的系統啊，同時也相當脆弱。真彌子「私、これでも18です。」實在是…為了15禁 game 的緣故才謊報年紀的嗎？另，為何シリア每次都撞到小次郎？依那機率，他可能每天都會撞個十來次吧。
音樂普通。動畫及 CG 方面，表現沒有 TFA 那麼好（那當然的），而且多了不少那時流行的卡通味。之後的作品現實感比較重，像是關起櫃子時櫃子就不會歪曲，搞怪動作也比較少（笑）。
這系列的本命越來越像是那女人了，小次郎的重要性漸漸變輕，所知的「內幕」也漸漸變少的樣子（嗚）；下一作不知如何。
