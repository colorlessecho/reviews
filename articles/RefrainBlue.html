Refrain Blue
簡介 這應該也是個 Jwin game 吧，否則我不會留到現在才玩。不過，因為某些原因終於不得不將 Cwin 還原，所以…在我的執念下，硬是在 cwin 平臺灌完リフレインブルー。細節不提，不過若是初心者，建議改改 Install.exe，或是灌到 link 出問題時 Ctrl+Alt+Del 將 Install.exe 關掉，直接試試下面的 reg 檔（別問我，我沒試過…）：
只是，因為我 cwin 支援較多（DirectX 吧），結果效果竟然比 Jwin 好…-_-;あ、そうそう、我在 Jwin 上沒灌 DirectX。-_-;;用 CWIN 時可以邊開漢字通。這 game system 最令我詬病的就是只有 full window mode，不過大概也虧了這個，看不到漢字通擾人的視窗。

看得出這是個「最終結果」相當龐大的作品。不說用不用心，光是那一大串 staff 就相當嚇人了。要是用了這麼多人還弄不出好東西，那才是…^^
感覺上有點像 KANON 般，有些 game 會安排些神祕的設定，就像溫馨版的推理劇場一樣，讓人一時摸不著頭腦，而在劇情進展中漸漸明朗化。KANON やリフレインブルーまではそーゆーもんだ。對於這樣的劇本，我到現在還是相當的沒抵抗力。^^;
它也有些特色為人稱道的，最大的是一定要 clear 了前五人才能達到 TRUE END - 深景。想節省時間一次解決「傳說中的女主角」（總有點…あゆあゆ的感覺）只好靠別人的 save 檔了。不過我是全玩的，所以沒差。再怎麼說，這劇情比起其他普通的 game 實在是相當好的。而且（只是，）總覺得還相當耗時，就像 staff 決心讓玩家看完八九成劇本方肯罷手。而我又相當鐵齒，非得每句臺詞都聽（有時還倒帶），加上還有川奈由織、早瀨雫這樣說話不是比普通人慢一倍（穏か、穏か、もどかしい…而且，找的配音員感覺不真的那麼溫柔。她的劇情也有點怪，我大概不會是這個善博。）就是吞吞吐吐（公式：×××...×××...）的女孩；結果玩起來就相當慢了。眼看期末考當前…還是玩完リフレインブルー再說吧。
エルフ的作品向來多所為人稱道，同、下級生就是；其他像最近的脫衣雀（我大概不會出手吧…？^^;）、之前伊頭家三兄弟シリーズ（遺作臭作鬼作）等，族繁不及備載。但，ノベル（小說）的就不多了。像之前提到那幾個，脫衣雀當然不用說，其他我也沒承認算是小說的。不過，リフレインブルー算是個不錯的，是想彰顯「我們 ELF 雖然不常作ノベル的作品，但一作就會有這樣的程度」嗎？^^;（氣のせい、氣のせい。）
音樂單獨拿出來聽其實不怎樣，但與劇情之配合度不錯。我加的★大概就是在配合度上了。語音也是。之前的 game 是沒注意，但リフレインブルー語音的意境配合度也是令我感心的。
リフレインブルー，其一大概是指不斷重複的 blue（切ない過去）吧。（clear 後注：果然，是繰り返す悲しみ的意思。）雖然沒有「在同一海岸偶遇，而各自懷抱著不同的哀傷」的感覺，但是其設定已帶給我相當不錯的意境。ただ、說是 blue，其實倒也不怎麼令人覺得哀怨。單獨說來，玩到 TRUE END 之前都沒令我淚をあふれる的情境。剛開始玩津賀島つぐみ時笑翻了，しようがないな、この子。玩到最後也感受不到她有啥悲哀，而「她的悲哀」和かね有啥關係我也不能理解。（被棄，所以缺錢；一天到晚只向錢看嗎？）早瀨雫，（剛玩還覺得「這傢伙佔語音資料的大小該不會是最小的吧」^^;）這是長頭髮的レイ嗎？髮色，不用說性情；連配音都相當類似(0_0)!! 而川奈由織好像也被美化成女神第二了。（其實深景也是）
最氣に入る的當然還是深景（心）。假如真有那麼やさしい的姐姐，我大概也會那麼作吧（是嗎…(笑)）。（啊…這就是二次元；他們老是搬些現實中不可能有的可愛妹妹出來…）畢竟，松永善博的部分已經相當接近現實的人性了（他的欲望與我同在）。劇本中臺詞部分比起其他 game 感覺上較不做作，而非「王子與公主從此過著幸福快樂的日子」的結局，事實上也比較接近現實，雖然有的濡れ場還是過於強求點（相處時間只有四五天而已嘛）。起碼，「我所認識的社會」（嗚～いつわりのかそうか）還沒到這麼簡單就交出處女的地步。不過，我似乎真的感覺到那種「大切」的心情（はずかしいけど、はじめてです。）。另外，假如想 H 就會落得 bad end，而 No H 才有好看 CG 的安排會如何呢…（笑）現在的 game 普遍都有處女情結（雖然這也得怪玩家沒錯…我也有…but…^_^;），假如來這招的話，說不定玩的人就更少了。
有幸在數天內馬不停蹄（用這個詞好嗎？^_^;每天十小時左右吧。-_-;可能沒那麼多）的啃完這個 game，感觸更深。尤其是最後 TRUE END 一次拚完…建議有時間的話，或是找個週末。花上五小時左右（從最後選項起我聽語音的速度。TRUE END 前半得 skip 個數十分…這是大總匯嗎？^^;skip 到煩，每個女孩都上場了。）就可以かんりょう了。當中，又是那種「現實と幻想が交差する」的劇情…既然「光を…」拿主角下手，那找天來將主角設定成精神病患如何（唔）。然後基礎設定，他所想像的一切，有難同當的朋友、溫馨關懷的家族（親人）、以至溫柔多情的愛人，全部都是自己的想像。（我…嗎）
也有點育成 game 的味道，藉助 game 來幫人體會某種情境，並告知應對的方法；教人如何克制自己，適應（接受）殘酷的（我很喜歡用這個詞^^;）現實。（真的有這種 game 嗎？）
另外，假如有人發現到的話，TRUE END 之前每個女孩幾乎都有加些特別的小遊戲。玩到第二三個才發現，懶得記了。不過這也是很有趣的。其他 game 製作者都還沒有這樣的手法。
若總歸一句話…這是個相當值得細細品味的作品（雖然音樂不怎樣）；即使在玩完後，我也還持續在那意境中好一陣子，就像看完一部不錯的小說（漫畫吧…）一般。或許有些人能在其中發現所謂「愛」的形式之一種（ONE 中也沒能感受到的…本當の愛か…）。當然，這是我必入手的 game 之一，值得。
