星空☆行星
還只玩過佳多奈。聽說這是其中最「本命」的劇本了。要不要再玩下去、仍在考慮中。
或許是過於期待了？其實，早在三月初星空就已存在硬碟中，只是玩過 prelude 後，我不確定是否有到 Menu 部分，（應該有吧…-_-）結果就這麼沈睡到玩完光を…、Phantom 以及 D 的現在。再開，是因為準備清硬碟了（汗）。
回想起當時之所以沒有怨念繼續苦撐下去，那是被可怕的人物 CG 以及學園派劇本打敗了（我向來對清純学園物語不抱好感，像 ToHeart 我就不會再玩第二次。^_^;）。人物 CG…真的和鈴がうたう日有拚哪，那種くそ度。完敗(^_^);;;;。
所以，我的琦想就只剩下原 Disk3 上面 Track1 的 OP 了，整個 game 值得我買的竟然只剩這條歌（！×！）OH-No~~~！
這樣也沒辦法，虧我還連買了兩次，因為 AUG 的壓壞了。（泣…ಥ_ಥ 大泣…死）好吧，就算這樣還是得來形容一下我心中的女神（？What is it?）是如何崩壞的。
首先，（CG 的差勁略過）發現 keyboard 除了 ↑/↓/Page Up/Page Down/Home/End/Enter ↵ 等非常有用之外，其他都是廢物。和 Phantom 一樣，對 keyboard 族是相當不便的。又是個 mouse 點一點就能過關的遊戲…不，也不能這麼說，應該說是他們在某方面相當努力，（在回溯以及 skip 方面相當不錯，能排上前三名。）但是其他又放得一塌糊塗。
故事，行進和 Lien 有點像，其實深度大概也是同等級的。（可是 CG，CG 啊啊啊～～）大致上一直選擇要攻略的女孩，然後是對其親近的選項就可以達到 end，也沒啥困難。我對這種方式的故事（聽說天藍の夏也是-_-）真是不太欣賞。能夠的話，是我自己能依照自己的性向能走一次的。つまり、是沒有誤差的，出來的就和現實中若我遇上這樣的環境以及事件，那我得到的結局就是故事中那樣子。這樣才是「模擬」啊。假如還有得選，那我大概永遠都是歸宅部吧。就因為現實中我們沒有機會碰到相似的情節，所以才要模擬啊。
沒事，我要說的，只是我不喜歡用這種方法（某個女孩會在某地出現）選來選去罷了。希望的是二選一，三選一那種「性向測驗」（爆）。
七年的別離，回想不起的事。又是個和 KANON 相似的構局。不過，深度是不同的，我習慣用 autoclick 讓他跑，不到一天就結束了，算滿短的吧。玩到下學期，只有一個感想：這 game 前半沒什麼，頂多算是普通無聊美少女ゲーム的程度吧。玩完頭腦裡根本不剩下什麼。（つまり、有玩和沒玩一樣的遊戲。對我來說，是不及格的。）重點一定擺在後半吧，為每個人安排了沈重的劇情；讓我來ゆっくり體會一下吧，嘿嘿。結果…沒有死的劇情，果然不能毀壞我嗎——
算了，就是這麼回事。CG 畫得是比人物來的好，不過也只有普通程度罷了。假如要全玩，skip 的部分太多了，而 system skip 的速度又不夠快，感覺上會浪費相當多時間。
老實說、應該有點同情分數吧。要打到90分以上，那就真得很勉強了。該不會我最近心情不好，所以打分數也特別嚴苛？(@_@)

終於將瞳解決了。總歸一句話。我不適合勵志故事。夢はかなえないこそ夢であり。
