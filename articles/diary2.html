讀書劄記 @ Colorless echo -the days never back-

青澀歲月終於到此結束，我人生的第二個階段大概也到此為止。
昨天晚上七八點媽突然打了電話，說因為颱風進來，希望晚上就準備一下全搬回來。事實上我心理都還沒準備好。更甭提裝箱啥的。就算如此，還是花一小時左右將有的沒的全清空了，剩下還當 server 和自己在用的那臺；當然，以及螢幕。看著寢室外那六箱東西，心中意外的是竟然沒原先想像的那麼多。或許是因為長久的寄宿加上沒有過交心的經驗，使我有了漂泊的感覺及習慣，生活就像旅人一般；沒有安定下來，把那地方當作自己真正的家那種感覺。就算住上一年，也像是在旅社一般。不過，換個方式說，我不管到那裡，也都能有種適應的假象，或者是說，固定的適應程度。畢竟到頭來終究是個客人，長時間的。
回想起當初搬來這裡時，那天由父母陪同，一整車雜七雜八的東西。我只是還以為雖然換了個地方，我還是能把那改變到像自己的家一般。但是，我似乎是奇特的，特立獨行而與人不相往來那種。所以，四年的同居生活很簡單的摧殘了我。厭惡流行音樂以及不能打入他人的生活圈一直是我與他人交往的一大瓶頸。一年級，新鮮人們都還互相尊重，當然也幸好和我同寢的還算這樣的傢伙。二三年，我一直是處於逃避的狀態。尤其是三年級，世界之大已經到達不能容我的地步。時常徘徊在寢室外一個人苦笑的我，就像是白痴一般。四年級倒還好，不過改不回來的已經沒辦法了，我帶過來的東西幾乎都是坐火車就簡單可以帶去的，而回來時則通常是大包小包。去掉電腦相關及母親最在意的衣服（所以我不能全帶回來），剩下大概一箱的東西吧，包括書。書？那種東西哪還需要。
爸來時我終於正常的和他交談。希望他們別以為是因這學期在學校出現啥奇蹟才讓我改變。高中一或二年級，在某次國文考試得到二十分的 shock 後，藉由這個機緣，我開始嘗試完全不和同學交談，同時斷絕了與家人的。きまぐれ、以我的情況或許可這麼說。大學與他人回復正常，但與家人則還是昨晚近十一點的事。的確，不應該是這樣的。心結，太多了。
回程，又是個下雨的日子。猶記得上次去高雄回來時，也是這麼下著雨，這麼的惆悵；不過總是又對自己的那份「人性」感到些許無奈。大學以來，我越來越有「我與人相處的技巧是基於意識下仿效他人而來」的感觸（和 FSS 主角一樣）。也因為我不在乎「禮儀」（或いは距離……不過我可不是可愛的小貓，而是死猴子。），而產生了基於貪婪守則的行事方法所衍生的人際關係交往術，使得很多人都覺得我很怪：不會占便宜過火了嗎？去掉理智的我到底會變成什麼樣子，我只能猜測不會被稱作人吧。話雖如此，上次喝酒喝到一定程度就想睡了，克制之下根本沒能到失去理智的程度。（室友還是說「這就是醉了」）我喝酒的頻率很低，但上上次也是這樣，從沒一次暴露出本性過。（可惜嗎）
好歹該作個總結吧，總之都來這麼一趟了。學前生活我真的不太記得了，僅存的斷片，我還算是個滿普通的傢伙。只是據後來的發展，很難想像我不是那種成天耍小聰明，一眼就知道不成大器的死小鬼。不，我應該還不到那程度，なぜなら、那幾幕已經沒有我和他人吵架的回憶了。所以，怎的也不能確定。
小學，不，幼稚園的生活是我惡性循環的開始。我從與他人的交往中，一步步習得「隔離」的技巧。習慣孤獨，以至樂於其中。我不知道（不是不知道？）「交心」的感覺，只是因為過於稀少，那漸漸地以成為回憶，假象，終於我完全不能確定自己是否真的拿出誠心待人過。我想了什麼？是的，就是現代人的通病。
室友說我令人感覺像是老人。更精確地說，是快死的人。自知向誰伸手都不會得到回應，而也不曾這麼做。的確，我一直想著的，或許你總有一天也會想到，只是希望那時候的眼神不會和我一樣落寞。
「對人的態度」總結到此。那我到底完成了什麼？依自證預言，套句我常說的話，結果就是我證明了我真的不算什麼，而且也從沒能做到什麼。
可悲嗎？悲傷不是在當事者眼中，而是出現在四周觀看的人。那又有何意義？成就，主要還是建立在他人的評價，而非在自己的努力之上。（固然，實力或許能改變他人的態度，但這就像錢不是萬能一樣。）
下午回來、大概整理了兩小時。雨、好像停了。
2001/6/23PM
才剛趕回來，又到高雄去了，為了將 server 跑起來。寫了個小程式，就這樣吧。不知何時要報到，感覺上卻像是多了個最後的暑假。換到了三樓前面，之前住後面。似乎經過了翻整修，多了好些東西，包括原來全家只剩我沒使用的冷氣。經濟好了，活夠久了，生活就會變好，似乎是很容易推得的道理。剛把東西搬進來，房間空蕩蕩的。地上堆滿雜七雜八的東西，一如我經常的習慣——擁擠時——才會去整理。現在三樓算全部都由我住了吧，雖然原來後面那間變成了儲藏室。經過那麼多年，終於把我換了個地方，主要還是因為三樓太熱的關係。長久下來，也練就了我平均較耐熱的功夫，雖然那似乎沒啥實用性。
一早二舅載回來時，雨大致已停了。不知西南氣流如何。昨晚餐就是肉粽，雖然明天才端午。不，就因如此，所以才在提前兩天作的吧。不過小時候好像都是當天作的……こんな記憶。高雄龍舟也是今天比，明天應該還有決賽吧？
2001/6/24AM
回來這幾天還真是啥事都沒做，一心不亂打電動。從回來那天起，我驚覺到舊的眼鏡竟然已不堪使用了。不能看電視，連在家都不太能看得清楚。反倒是新的那副，現在不管打電腦或是看二十公分內的東西都悲哀得好用。算了，再招也不會來臨的東西。
2001/6/26PM
在炎熱夏日的高層建築中，令人昏睡的浮動視線使我被擊沈在床上。沈重的意識，感覺到生活如同跑馬燈。到底夢是現實，或現實是夢都不太能分得清了。爸為了傳達「到下午六點在棋社」的訊息，爬了上來。雖然這並不是我遇過最難熬的一次，甚至只是在回來這幾天內，但他還是按下了除了回來那天試用外第二次，也是正式啟用的冷氣。因為某種已經接近自殘的生活習慣（本人並不承認），我還是持續的訓練自己耐熱。也因此，對於使用冷氣後我會不會更難回復，抱持著莫名其妙，似乎（應該說）是恐懼的感情。諷刺的是，耐冷的功夫靠棉被等就可以實現，但對於過於酷熱的環境，現代人幾乎都是靠空調以達成，所以這個家裡也只配備大衣、冷氣及電風扇。
這幾天硬碟壞軌，正努力的把東西解決掉。幸好失去的東西並不算太多，不過終究還是有些許遺憾。
2001/6/27PM
說也奇怪、家門前的郵箱蓋自從回來那天風雨交加後隔天就失蹤了。家人也沒有任何人說什麼，但近鄰幾家也就只有我們這樣，看起來實在相當滑稽。
2001/6/28PM
晚上參加了某社聚會，記得去年也參加過，擺明了是為某立委辦流水席（另外我可能也順便看過那立委幾次）。不過，兩次都令我感心的是，吃完馬上走人。宣傳是宣傳，強迫性並不高。那立委也相當配合，連飯都沒吃，在臉上明寫一排「我在趕場，煩我者死。」所以主辦人總是識相的從頭捧場到尾，十分鐘之內人就走了。因為種種跡象太過明顯，所以增加信心的也只有本來就已經相信的人——人只看到自己想看的。
媽總說三樓太熱，其實我也覺得這相當熱，尤其是這次終於搬來電腦之後。剛剛終於拿來一支溫度計一量，原來我一直生活在 35 °C 的環境中。晚上了，才好容易降到 34.5 °C，也沒差多少。難怪我冷氣再怎麼開，溫度再怎麼調，感覺都沒啥差別。
昨晚妹宣布被二一，壽命大概減了一年（雖然本人依然一副高飛車女的樣子）。我明天要去報到，情況也沒好多少。「能活的話才活下去」竟變成我現在的生活態度了。家中下一代這兩個不成材的都處於危機中。不過來自媽的壓力很大，我現在只等到能別讓他們傷心，然後又能讓他們從此不用管我的事發生。再之後，就準備破滅，完結自己了。
下午又買了新片，雖然怎麼看都是被黑了錢，不過買了就買了，又沒辦法退貨。看來未來這幾天，得趕緊把硬碟清空了。欸……還有五十片以上吧。不少的樣子。
今天的日記實用性真高，純記事，根本沒啥自己的意見。也罷，意見會越來越少吧，因為出了社會就懶得去想了。「能不去想就別想吧」也是我悲哀的態度之一。
2001/7/1PM
夏天總是炎熱的。據這幾天的觀察，普通的日子總有三十四五度（不論白天黑夜），普通有電風扇之下還能正常過活。開冷氣也只能降到30度左右，不過其實32度我就能活得很好了。只是，若有三十六度以上，我恐怕會撐不住。這麼說來其實也算普通吧，我的情況。因為總是處於這樣的溫度之下，所以三樓我總是保持最通風狀態，附加電風扇。大前天吧，颱風來。沒下雨，不過風不小。那幾天內，氣溫有降到30度，不過還是有人開冷氣。那時候是不知道三樓是不是最熱的，（因為總是保持通風，所以有時還會比樓下涼。）不過看來有些人三十二三度就撐不住了啊。我想，我該嚐嚐四十度的感覺。那的確會熱死人吧。
2001/7/7PM

2001/7/12PM
下午去載媽，一到恰好一點半碰上到兩點的防空演習。本來心情是很糟，不過因為想到那裡有佛學書，既然不能騎車上路，不如就在那看一看。這次花了四十分左右，讀的是達賴喇嘛的著述。（西藏佛法簡介？見 2004/7/21 2:28。）萬物相互牽引，萬相為空，因此無常。執著，當只為眼前的型態；故執著心無用，一旦形勢改變，徒增苦惱。「空」是指像是緊緊交纏的線，當一有變動，方才看到的相就隨之改變，因此其相沒有「永恆」。……好像是這樣吧，似乎和我之前以為的不一樣。相異處在我重在「相之空」，而他們重在「互動」（相互牽引）。嗯，是觀念上滿大的差別。而且，他們要身體力行，我只要知道就好。所以，我是封閉的（只看到自己），而他們提倡交流。我甚至在中間還看到社會心理學的東西，就是基模和無明，「對萬事萬物的錯誤認知」（笑）；ま、即使是他們，也有點共產主義（理想化）啊，人有許多事是不能「因為這麼做是對的，所以應該這麼做。」的。幸好他們修行的恰好就是以那些教條為根基，所以才能撐那麼久。（不守戒律，那就別修行了！——踢出去。）不以人之道德性為基本的共產主義就沒那麼幸運了。（就算沒道德，總還是人民；不過又沒法管他。）喔、對了，他們恐怕會誤以為我就是他們所謂「斷滅空」、頑空，畢竟，我只看到「空」嘛（笑）。

2001/8/4AM
這個月來最重要的工作，投入最多的是在ビジュアルノベル。play, 寫心得之類的，看看能不能免疫（笑）。又是好久沒更新了。昨下午下了場小雨，不過因為太晚了（傍晚吧），還是很熱。晚上因為勞累而提早三四小時上床。或許因為違反了一個月來的習慣，聽著音樂睡睡醒醒大概也三四小時。結果今早七點起來，一點也看不出來昨晚有早睡過。早上還是去社教館。最近和那裡某人似乎交流有點過於頻繁了，不過那也是長久的孽緣，人情反正就是這麼回事吧。說要斷也不是那麼好斷的。現在情況看來就像隔壁清純的大姊姊和小弟弟的關係，所以 let it be 吧。
2001/8/11PM
最後一個暑假到這禮拜結束。接下來盡是無明的生活。好像我的生活也從來沒明過。妹經過七八天，昨晚坐統聯從臺北回來，凌晨到家。幾天來下午偶爾會下雨，但對氣溫的降低都沒啥幫助。
回思1：因為玩 game，Discovery 也少看了。一兩禮拜前聽到腦部在回想到重大事件時會有 300 波，有些人想因此作法律性判決。只是我想，想過並模擬過，或是牽扯出其他聯想時，可能無中生有；而訓練或是重大打擊應該也有機會導致想過的事變調吧。假如真的如此作，又有新的悲劇題材可以寫了。
回思2：未親身經歷就是不瞭解。但媒體大肆報導下，多少可以感受到一種世界末日的感覺。桃芝颱風的事後報導中，不經意聽到一句「那地方本來就不適合住人」，而遷村的消息。星期日 Discovery 如此介紹墨西哥大地震：首都在哪都好，就是不該在那地方。我們是不是真的有些牽強的要求大自然配合了？假如有一天旅人回到故鄉，只見寂寥的遺骸，他會推論人們因疾病滅絕，或是為環境擊倒而集體遷徙？
2001/8/15AM
早上一直拉肚子，該不會是昨晚吃的桑葚？……不過並沒吃那麼多啊，該不會連對桑葚都過敏了。暑假終於結束，爸媽也起程到大陸去。他們才剛走，妹就招待朋友到家裡；好像壓抑很久了。「這兩天我有朋友來，下去記得穿衣服喔。」
放假了，想想看，哪裡是比較好玩的？好久沒回鄉下探望爸媽（或祖父母）了，趁今天下午沒事去看看吧？……你是否也曾這麼想過？的確是這樣，短暫的森林浴，溪頭、走馬瀨踏青之旅，總是可以紓解人的煩躁心情。我到國中前，父母帶我出遊的次數，或許因為經濟能力允許，加上父親的習慣，大致在一般人以上。但，十年來，我幾乎不曾再到高山欣賞美麗森林、或到某景致宜人的度假勝地戲水。算算，不超過三十次吧，只算最近四年更只有一次；且這些大部分還是像畢業旅行等非自己意願的例行公事。青草綠地、湛藍天空的事，已經好久未曾在我腦海中浮現了。甚至，連僅存的直系祖父母輩也是在暌違六七年後才在前不久表姐的婚禮中相見。這看起來像是自虐，「想要就去啊，有什麼阻止你了？還在這唉唉叫。」但，不想、不能出去的意念卻如同詛咒般，使我一直拒絕著任何的邀約。我知道的只是，就算我出到外面，也無法感受到完全放鬆的心情，就如同我在人死時甚至可以笑出來一樣。無感覺。努力的保持笑容以使自己姑且快樂。我的生活就這麼漸漸陷入一種無力救贖的狀態。而且，是我自己想要這麼做。

假如以我從此書中所得到的訊息來評論，（當然他們一定會說我的想法是錯的，雖然不知道是不是大錯特錯。）基本上，我所謂的「空」和他們心底的是不一樣的。我所謂的「空」是指世事及所有相的飄渺而はかない，但他們的空則是拒絕任何的存在：原子不存在，樹木不存在，月球及地球以至宇宙都不存在。他們或許是假設「他們存在」，卻得到矛盾，因而一切應是不存在的。那倒是滿有趣的：他們「假設中的」東西照他們所言是不存在，但事物本性是否一定正如他們所假設，可只有他們能打包票。就算真的一切不存在，又何來起波瀾？萬物會有「痛苦」「幸福」，他們會「想到」一切是空，本來就代表因為有所不是空的東西造成波瀾，因此本來「空」的狀態下不會出現的思想、五蘊全都出現。「空」的本質假如說成萬物相對而無「絕對」之存在還不會有太大矛盾，但連「存在」本身都被否定的話，那麼佛陀是不存在的，他們也是不存在的。既然佛陀也不存在，他們也不存在，何以會有「他們」來記得個「佛陀」？他們是夢幻，佛陀也是夢幻。所以這全是我腦中浮現的一場夢。但我之所以會浮現夢，「全是空」理論可完全不能解釋。依「全是空」理論，我「絕對不會」，而不是「不應該」產生這些念頭。這，可與事實相悖了；所以說矛盾？
但是他們的結論卻有部分與我相通之處。假的假設即使推出真的結論，這也不能表示那假設是對的。順帶一提，應成派主張「四不生」卻說是無任何主張，根本就是咬文噘字的錯誤。就算完全無任何主張，那麼他們的戰場全是對方的地盤，游牧民族第二；那也就無任何本身立場批判人。所以，他們只能「以子之矛攻子之盾」，那叫做無建設性破壞，因為除了破壞並不能帶來任何建議。話說回來，既然都是空，何必挑起爭端否決他人？爭了是空，不爭也是空。大概就是因為這些基礎理論的瑕疵，才造成同是學佛，行為萬萬千吧。既然都是對的，何必還站出來大聲說「你錯了」或「我才對」？他們的平和，是平和的假象，以寬宏及修養來掩飾觀念上的爭執。
昨晚看過 Discovery「智障天才」後，卻不曉得自己到底是憂鬱症，或是自閉配上學者症候群（這個不太可能，我沒天才。）？唯一知道的是，就算我是，也是非常輕微的，而我那稍微的小聰明，應該不過只是這種缺陷下的副產物吧。因為社交上的缺陷，造就某方面的成就。而當人際關係越是順手，成就卻越來越低了。我恐怕不是真正的「聰明（聰慧）」吧。
